# 1.11.3 创建自己的Content Provider

“如果要让自己App的数据能被其他App访问，就需要自定义Content Provider，”黛琳说道。

## 1.11.3.1 声明Provider

“在Manifest中声明，”黛琳讲解道。

```xml
<provider
    android:name=".MyProvider"
    android:authorities="com.example.provider"
    android:exported="true"
    android:grantUriPermissions="true" />
```

## 1.11.3.2 实现Provider

“实现ContentProvider的六个核心方法，”黛琳继续说道。

```kotlin
class MyProvider : ContentProvider() {
    
    private lateinit var dbHelper: MyDatabaseHelper
    
    // 初始化
    override fun onCreate(): Boolean {
        dbHelper = MyDatabaseHelper(context)
        return true
    }
    
    // 查询
    override fun query(uri: Uri, projection: Array<String>?, 
                       selection: String?, selectionArgs: Array<String>?,
                       sortOrder: String?): Cursor? {
        val db = dbHelper.readableDatabase
        val cursor = db.query("users", projection, selection, 
                            selectionArgs, null, null, sortOrder)
        cursor?.setNotificationUri(context?.contentResolver, uri)
        return cursor
    }
    
    // 插入
    override fun insert(uri: Uri, values: ContentValues?): Uri? {
        val db = dbHelper.writableDatabase
        val id = db.insert("users", null, values)
        if (id > 0) {
            context?.contentResolver?.notifyChange(uri, null)
            return ContentUris.withAppendedId(uri, id)
        }
        return null
    }
    
    // 更新
    override fun update(uri: Uri, values: ContentValues?, 
                       selection: String?, selectionArgs: Array<String>?): Int {
        val db = dbHelper.writableDatabase
        val count = db.update("users", values, selection, selectionArgs)
        if (count > 0) {
            context?.contentResolver?.notifyChange(uri, null)
        }
        return count
    }
    
    // 删除
    override fun delete(uri: Uri, selection: String?, 
                       selectionArgs: Array<String>?): Int {
        val db = dbHelper.writableDatabase
        val count = db.delete("users", selection, selectionArgs)
        if (count > 0) {
            context?.contentResolver?.notifyChange(uri, null)
        }
        return count
    }
    
    // MIME类型
    override fun getType(uri: Uri): String {
        return when (uriMatcher.match(uri)) {
            USERS -> "vnd.android.cursor.dir/vnd.$AUTHORITY.users"
            USER_ID -> "vnd.android.cursor.item/vnd.$AUTHORITY.users"
            else -> throw IllegalArgumentException("Unknown URI: $uri")
        }
    }
}
```

## 1.11.3.3 专业技术总结

本章我们学习了如何创建Content Provider。

**核心要点：**

1. **声明Provider** - Manifest配置
2. **实现六个方法** - CRUD + getType + onCreate
3. **notifyChange** - 通知数据变化
4. **URI匹配** - 区分不同请求
5. **MIME类型** - vnd.android.cursor.dir/item

**注意事项：**

- exported属性控制是否导出
- grantUriPermissions控制权限
- 线程安全很重要
