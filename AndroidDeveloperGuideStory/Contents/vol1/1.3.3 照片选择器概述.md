# 1.3.3 照片选择器

"Android 13引入系统级照片选择器，"黛琳介绍道。

## 使用

```kotlin
val intent = Intent(MediaStore.ACTION_GET_CONTENT).apply {
    type = "image/*"
    putExtra(Intent.EXTRA_ALLOW_MULTIPLE, true)
}
startActivityForResult(intent, REQUEST_CODE)
```

## 优点

- 不需要存储权限
- 用户选择特定照片
- 保护隐私
