# 1.3.5 帐篷里的文档时光

帐篷里，黛琳正在整理她的学习笔记。洛FR凑过去看：“姐姐，你的笔记能分享给我吗？”

“当然可以！”黛琳笑道，“咱们今天就来说说怎么访问共享存储中的文档。”

## 1.3.5.1 使用SAF访问文档

“SAF就是存储访问框架（Storage Access Framework），”黛琳解释道，“它让你可以访问设备上任何位置的文件。”

```kotlin
// 打开文档
fun openDocument() {
    val intent = Intent(Intent.ACTION_OPEN_DOCUMENT).apply {
        addCategory(Intent.CATEGORY_OPENABLE)
        type = "application/pdf"  // PDF
        // 或者 "*/*" 选择所有文件
        // 或者 "image/*" 图片
    }
    startActivityForResult(intent, REQUEST_CODE)
}
```

## 1.3.5.2 处理文档结果

“选择文件后，要处理返回的URI，”黛琳继续说道。

```kotlin
override fun onActivityResult(requestCode: Int, resultCode: Int, data: Intent?) {
    if (requestCode == REQUEST_CODE && resultCode == RESULT_OK) {
        data?.data?.let { uri ->
            // 读取文件内容
            readDocument(uri)
            
            // 获取持久权限
            val takeFlags = Intent.FLAG_GRANT_READ_URI_PERMISSION
            contentResolver.takePersistableUriPermission(uri, takeFlags)
        }
    }
}

private fun readDocument(uri: Uri) {
    try {
        contentResolver.openInputStream(uri)?.use { inputStream ->
            // 读取内容
            val bytes = inputStream.readBytes()
            // 处理文件
        }
    } catch (e: Exception) {
        Log.e("Document", "Error reading", e)
    }
}
```

## 1.3.5.3 创建新文档

“如果要创建新文档呢？”洛FR问。

“用ACTION_CREATE_DOCUMENT，”黛琳说道。

```kotlin
// 创建新文档
fun createDocument() {
    val intent = Intent(Intent.ACTION_CREATE_DOCUMENT).apply {
        addCategory(Intent.CATEGORY_OPENABLE)
        type = "text/plain"  // 文本文件
        putExtra(Intent.EXTRA_TITLE, "my_notes.txt")  // 默认文件名
    }
    startActivityForResult(intent, CREATE_REQUEST_CODE)
}

override fun onActivityResult(requestCode: Int, resultCode: Int, data: Intent?) {
    if (requestCode == CREATE_REQUEST_CODE && resultCode == RESULT_OK) {
        data?.data?.let { uri ->
            // 写入内容
            contentResolver.openOutputStream(uri)?.use { outputStream ->
                outputStream.write("Hello, World!".toByteArray())
            }
        }
    }
}
```

## 1.3.5.4 打开文件夹

“还可以打开文件夹，让用户选择里面的文件，”黛琳补充道。

```kotlin
// 打开文件夹
fun openFolder() {
    val intent = Intent(Intent.ACTION_OPEN_DOCUMENT_TREE)
    startActivityForResult(intent, TREE_REQUEST_CODE)
}

override fun onActivityResult(requestCode: Int, resultCode: Int, data: Intent?) {
    if (requestCode == TREE_REQUEST_CODE && resultCode == RESULT_OK) {
        data?.data?.let { treeUri ->
            // 获取整个文件夹的访问权限
            val takeFlags = Intent.FLAG_GRANT_READ_URI_PERMISSION or
                          Intent.FLAG_GRANT_WRITE_URI_PERMISSION
            contentResolver.takePersistableUriPermission(treeUri, takeFlags)
            
            // 列出文件夹内容
            listFolderContents(treeUri)
        }
    }
}
```

## 1.3.5.5 专业技术总结

本章我们学习了如何使用SAF访问文档。

**核心要点：**

1. **SAF提供统一访问** - 不需要遍历文件路径
2. **ACTION_OPEN_DOCUMENT** - 打开现有文件
3. **ACTION_CREATE_DOCUMENT** - 创建新文件
4. **ACTION_OPEN_DOCUMENT_TREE** - 打开文件夹
5. **需要处理持久权限** - 长期访问需要

**Intent类型：**

| Intent | 用途 |
|--------|------|
| ACTION_OPEN_DOCUMENT | 打开文件 |
| ACTION_CREATE_DOCUMENT | 创建文件 |
| ACTION_OPEN_DOCUMENT_TREE | 打开文件夹 |
