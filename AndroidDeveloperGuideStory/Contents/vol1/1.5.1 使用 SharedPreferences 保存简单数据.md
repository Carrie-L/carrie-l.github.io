# 1.5.1 篝火旁的设置宝箱

夜晚的篝火旁，洛FR正在摆弄手机：“黛琳姐姐，我想记住用户的选择，比如他喜欢什么主题、上次登录的时间...这些怎么保存？”

黛琳笑道：“这就要用到SharedPreferences了！”

## 1.5.1.1 什么是SharedPreferences

“SharedPreferences就像是一个小的'设置清单'，”黛琳解释道，“专门用来存简单的小数据，比如用户的设置、偏好等。”

**适合存储：**
- 用户名
- 主题选择（深色/浅色）
- 开关设置
- 上次登录时间

**不适合存储：**
- 大量数据 → 用数据库
- 复杂结构 → 用Room

## 1.5.1.2 基本用法

“用法非常简单，”黛琳说道。

```kotlin
// 保存数据
fun saveSettings() {
    val prefs = getSharedPreferences("my_settings", MODE_PRIVATE)
    
    prefs.edit().apply {
        putString("username", "洛芙")
        putBoolean("dark_mode", true)
        putInt("font_size", 16)
        putLong("last_login", System.currentTimeMillis())
        apply()  // 异步保存
    }
}

// 读取数据
fun loadSettings(): Settings {
    val prefs = getSharedPreferences("my_settings", MODE_PRIVATE)
    
    return Settings(
        username = prefs.getString("username", "") ?: "",
        darkMode = prefs.getBoolean("dark_mode", false),
        fontSize = prefs.getInt("font_size", 14),
        lastLogin = prefs.getLong("last_login", 0)
    )
}
```

## 1.5.1.3 模式选择

“SharedPreferences有几种模式，”黛琳介绍道。

| 模式 | 说明 | 注意 |
|------|------|------|
| MODE_PRIVATE | 只有自己的App能访问 | 推荐使用 |
| MODE_WORLD_READABLE | 其他App可读 | 已废弃 |
| MODE_WORLD_WRITEABLE | 其他App可写 | 已废弃 |

```kotlin
// 推荐用法
val prefs = getSharedPreferences("settings", Context.MODE_PRIVATE)

// 还可以用PreferenceManager获取默认偏好
val defaultPrefs = PreferenceManager.getDefaultSharedPreferences(context)
```

## 1.5.1.4 commit vs apply

“保存数据有两个方法：commit和apply，”黛琳重点说道。

```kotlin
// commit() - 同步保存，返回boolean表示是否成功
val success = prefs.edit().putString("name", "洛芙").commit()

// apply() - 异步保存，没有返回值
prefs.edit().putString("name", "洛芙").apply()

// 区别：
// commit - 同步，会阻塞，保存到磁盘
// apply - 异步，不阻塞，写入内存后立即返回
```

## 1.5.1.5 监听变化

“如果想监听设置的变化，可以用OnSharedPreferenceChangeListener，”黛琳补充道。

```kotlin
// 注册监听器
prefs.registerOnSharedPreferenceChangeListener { prefs, key ->
    Log.d("Settings", "Key changed: $key")
}

// 变化时回调
// key是变化的键名

// 取消注册
prefs.unregisterOnSharedPreferenceChangeListener { ... }
```

## 1.5.1.6 专业技术总结

本章我们学习了SharedPreferences。

**核心要点：**

1. **适合简单数据** - 设置、偏好等小数据
2. **key-value存储** - 键值对形式
3. **两种保存方式** - commit同步，apply异步
4. **推荐MODE_PRIVATE** - 保证安全
5. **可以监听变化** - OnSharedPreferenceChangeListener

**使用场景：**

| 场景 | 推荐 |
|------|------|
| 用户设置 | SharedPreferences |
| 简单状态 | SharedPreferences |
| 大量数据 | Room |
| 复杂结构 | Room |
