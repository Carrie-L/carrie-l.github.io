# 1.6.10 创建数据库视图

"视图是预定义的查询，"黛琳介绍道。

```kotlin
@DatabaseView(
    viewName = "user_with_book_count",
    value = """
        SELECT users.id, users.name, COUNT(books.id) as book_count
        FROM users
        LEFT JOIN books ON users.id = books.authorId
        GROUP BY users.id
    """
)
data class UserWithBookCount(
    val id: Long,
    val name: String,
    val bookCount: Int
)

@Database(entities = [User::class, Book::class], views = [UserWithBookCount::class])
abstract class AppDatabase : RoomDatabase() {
    abstract fun userDao(): UserDao
}
```
