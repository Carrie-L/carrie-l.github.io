# 1.6.2 Room实体定义

"Room用Entity定义数据表，"黛琳介绍道。

## 基本实体

```kotlin
@Entity(tableName = "users")
data class User(
    @PrimaryKey(autoGenerate = true)
    val id: Long = 0,
    
    val name: String,
    val email: String,
    val age: Int = 0
)
```

## 常用注解

```kotlin
@Entity(tableName = "products")
data class Product(
    @PrimaryKey
    val sku: String,
    
    @ColumnInfo(name = "product_name")
    val name: String,
    
    val price: Double,
    
    @ColumnInfo(index = true)
    val category: String,
    
    val inStock: Boolean = true,
    
    val createdAt: Long = System.currentTimeMillis()
)
```

## 复杂类型

```kotlin
@Entity
data class Order(
    @PrimaryKey
    val id: Long,
    val userId: Long,
    val items: String, // JSON序列化
    val status: OrderStatus,
    val totalAmount: Long // 分（整数）
)

enum class OrderStatus {
    PENDING, CONFIRMED, SHIPPED, DELIVERED, CANCELLED
}
```
