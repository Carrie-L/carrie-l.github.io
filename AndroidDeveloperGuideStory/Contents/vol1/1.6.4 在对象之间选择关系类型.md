# 1.6.4 Room关系类型

"Room支持多种关系类型，"黛琳介绍道。

## 一对一关系

```kotlin
@Entity(tableName = "users")
data class User(
    @PrimaryKey val id: Long,
    val name: String
)

data class UserWithProfile(
    @Embedded val user: User,
    @Relation(
        parentColumn = "id",
        entityColumn = "userId"
    )
    val profile: UserProfile
)
```

## 一对多关系

```kotlin
@Entity(tableName = "authors")
data class Author(
    @PrimaryKey val id: Long,
    val name: String
)

@Entity(tableName = "books")
data class Book(
    @PrimaryKey val id: Long,
    val title: String,
    val authorId: Long
)

data class AuthorWithBooks(
    @Embedded val author: Author,
    @Relation(
        parentColumn = "id",
        entityColumn = "authorId"
    )
    val books: List<Book>
)
```
