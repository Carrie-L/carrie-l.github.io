# 1.6.8 嵌套关系

"嵌套关系就是关系里面有关系，"黛琳讲解道。

```kotlin
// 作者 -> 书 -> 章节
data class AuthorWithBooksWithChapters(
    @Embedded val author: Author,
    @Relation(
        entity = Book::class,
        parentColumn = "id",
        entityColumn = "authorId"
    )
    val books: List<BookWithChapters>
)

data class BookWithChapters(
    @Embedded val book: Book,
    @Relation(
        parentColumn = "id",
        entityColumn = "bookId"
    )
    val chapters: List<Chapter>
)
```
