# 1.8.2 春天的分享时光

“黛琳姐姐，我想把App里的内容分享到微信或者别的App，该怎么做？”洛FR问道。

黛琳笑道：“这就要用到Android的分享功能了！”

## 1.8.2.1 使用Intent发送数据

“在Android中，分享是通过Intent实现的，”黛琳解释道。

```kotlin
// 分享文本
fun shareText(text: String) {
    val intent = Intent(Intent.ACTION_SEND).apply {
        type = "text/plain"
        putExtra(Intent.EXTRA_TEXT, text)
    }
    
    // 弹出分享选择器
    val chooser = Intent.createChooser(intent, "分享到")
    startActivity(chooser)
}
```

## 1.8.2.2 分享其他类型

“除了文本，还可以分享其他类型的数据，”黛琳继续说道。

```kotlin
// 分享图片
fun shareImage(imageUri: Uri) {
    val intent = Intent(Intent.ACTION_SEND).apply {
        type = "image/*"
        putExtra(Intent.EXTRA_STREAM, imageUri)
        addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
    }
    
    startActivity(Intent.createChooser(intent, "分享图片"))
}

// 分享文件
fun shareFile(fileUri: Uri, mimeType: String) {
    val intent = Intent(Intent.ACTION_SEND).apply {
        type = mimeType
        putExtra(Intent.EXTRA_STREAM, fileUri)
        addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
    }
    
    startActivity(Intent.createChooser(intent, "分享文件"))
}
```

## 1.8.2.3 分享多个内容

“如果要分享多个内容呢？”伊莎问。

“那就用ACTION_SEND_MULTIPLE，”黛琳说道。

```kotlin
// 分享多张图片
fun shareMultipleImages(uris: List<Uri>) {
    val intent = Intent(Intent.ACTION_SEND_MULTIPLE).apply {
        type = "image/*"
        putParcelableArrayListExtra(Intent.EXTRA_STREAM, ArrayList(uris))
        addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
    }
    
    startActivity(Intent.createChooser(intent, "分享多张图片"))
}
```

## 1.8.2.4 专业技术总结

本章我们学习了如何向其他应用发送数据。

**核心要点：**

1. **使用Intent.ACTION_SEND** - 发送单个内容
2. **使用ACTION_SEND_MULTIPLE** - 发送多个内容
3. **设置正确的MIME类型** - text/plain、image/*等
4. **添加权限flag** - FLAG_GRANT_READ_URI_PERMISSION
5. **使用选择器** - createChooser

**常见MIME类型：**

| 类型 | MIME |
|------|------|
| 文本 | text/plain |
| 图片 | image/* |
| 视频 | video/* |
| 任意 | */* |
