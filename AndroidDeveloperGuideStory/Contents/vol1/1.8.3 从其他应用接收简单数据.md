# 1.8.3 接收春天的分享

“刚才学了怎么分享，那怎么接收别人分享的数据呢？”洛FR问道。

黛琳说道：“配置Intent Filter就可以接收了！”

## 1.8.3.1 配置Intent Filter

“在Manifest中配置Intent Filter，”黛琳讲解道。

```xml
<activity
    android:name=".MainActivity"
    android:exported="true">
    
    <intent-filter>
        <action android:name="android.intent.action.MAIN" />
        <category android:name="android.intent.category.LAUNCHER" />
    </intent-filter>
    
    <!-- 接收文本分享 -->
    <intent-filter>
        <action android:name="android.intent.action.SEND" />
        <category android:name="android.intent.category.DEFAULT" />
        <data android:mimeType="text/plain" />
    </intent-filter>
    
    <!-- 接收图片分享 -->
    <intent-filter>
        <action android:name="android.intent.action.SEND" />
        <category android:name="android.intent.category.DEFAULT" />
        <data android:mimeType="image/*" />
    </intent-filter>
    
</activity>
```

## 1.8.3.2 处理收到的数据

“在Activity中处理收到的Intent，”黛琳继续说道。

```kotlin
class MainActivity : AppCompatActivity() {
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // 处理接收到的数据
        handleIntent(intent)
    }
    
    override fun onNewIntent(intent: Intent?) {
        super.onNewIntent(intent)
        handleIntent(intent)
    }
    
    private fun handleIntent(intent: Intent?) {
        when (intent?.action) {
            Intent.ACTION_SEND -> {
                if (intent.type == "text/plain") {
                    // 接收文本
                    val text = intent.getStringExtra(Intent.EXTRA_TEXT)
                    handleText(text)
                } else if (intent.type?.startsWith("image/") == true) {
                    // 接收图片
                    val imageUri = intent.getParcelableExtra<Uri>(Intent.EXTRA_STREAM)
                    handleImage(imageUri)
                }
            }
            Intent.ACTION_SEND_MULTIPLE -> {
                if (intent.type?.startsWith("image/") == true) {
                    // 接收多张图片
                    val uris = intent.getParcelableArrayListExtra<Uri>(Intent.EXTRA_STREAM)
                    handleMultipleImages(uris)
                }
            }
        }
    }
    
    private fun handleText(text: String?) {
        Log.d("Share", "Received text: $text")
    }
    
    private fun handleImage(uri: Uri?) {
        Log.d("Share", "Received image: $uri")
    }
    
    private fun handleMultipleImages(uris: List<Uri>?) {
        Log.d("Share", "Received ${uris?.size} images")
    }
}
```

## 1.8.3.3 专业技术总结

本章我们学习了如何接收其他应用发送的数据。

**核心要点：**

1. **配置Intent Filter** - 在Manifest中声明
2. **action是ACTION_SEND** - 接收分享
3. **处理onNewIntent** - Activity重启时也能接收
4. **检查MIME类型** - 区分不同类型的数据
5. **处理多内容** - ACTION_SEND_MULTIPLE

**常见MIME类型配置：**

```xml
<data android:mimeType="text/plain" />
<data android:mimeType="image/*" />
<data android:mimeType="video/*" />
```
