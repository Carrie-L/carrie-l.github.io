# 1.8.4 Direct Share

"Direct Share让分享更快捷，"黛琳介绍道。

## 使用ShortcutManager

```kotlin
class MainActivity : AppCompatActivity() {
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // 创建分享目标
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.M) {
            createShareShortcut()
        }
    }
    
    @RequiresApi(Build.VERSION_CODES.M)
    private fun createShareShortcut() {
        val shortcutManager = getSystemService(ShortcutManager::class.java)
        
        val intent = Intent(this, ShareReceiverActivity::class.java).apply {
            action = Intent.ACTION_SEND
            putExtra("contact", "user123")
        }
        
        val shortcutInfo = ShortcutInfoCompat.Builder(this, "share_contact")
            .setShortLabel("分享给朋友")
            .setLongLabel("分享给指定朋友")
            .setIntent(intent)
            .setCategories(setOf("share"))
            .build()
        
        shortcutManager.pushDynamicShortcut(shortcutInfo)
    }
}
```

## 声明Shortcut

```xml
<activity android:name=".ShareReceiverActivity">
    <intent-filter>
        <action android:name="android.intent.action.SEND" />
        <category android:name="android.intent.category.DEFAULT" />
    </intent-filter>
    
    <meta-data
        android:name="android.servicechooserchooser_target_service"
        android:value=".ChooserTargetService" />
</activity>
```

## 总结

Direct Share让用户快速分享到指定目标。
