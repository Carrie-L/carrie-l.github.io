# 1.9.1 文件共享概述

"文件共享让应用可以安全地分享文件，"黛琳介绍道。

## 核心概念

1. **FileProvider** - 生成content:// URI
2. **URI权限** - 临时或永久
3. **MIME类型** - 正确设置文件类型

## URI权限

```kotlin
// 临时权限（默认）
intent.flags = Intent.FLAG_GRANT_READ_URI_PERMISSION

// 永久权限
contentResolver.takePersistableUriPermission(uri, Intent.FLAG_GRANT_READ_URI_PERMISSION)
```

## 安全原则

- 最小权限原则
- 只授予需要的权限
- 定期检查导出的组件
