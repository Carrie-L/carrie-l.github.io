# 1.9.3 共享文件给其他App

“如果想分享文件（比如PDF、文档），该怎么处理？”洛FR问道。

“那就要用到FileProvider了，”黛琳说道。

## 1.9.3.1 为什么用FileProvider

“在Android 7.0之后，不能直接分享文件的URI，”黛琳解释道，“要用FileProvider来生成安全的URI。”

```kotlin
// ❌ 错误方式（Android 7.0+会崩溃）
val uri = Uri.fromFile(file)

// ✅ 正确方式：使用FileProvider
val uri = FileProvider.getUriForFile(
    context,
    "${context.packageName}.fileprovider",
    file
)
```

## 1.9.3.2 配置FileProvider

“首先在Manifest中声明FileProvider，”黛琳继续说道。

```xml
<provider
    android:name="androidx.core.content.FileProvider"
    android:authorities="${applicationId}.fileprovider"
    android:exported="false"
    android:grantUriPermissions="true">
    
    <meta-data
        android:name="android.support.FILE_PROVIDER_PATHS"
        android:resource="@xml/file_paths" />
        
</provider>
```

## 1.9.3.3 配置路径

“在res/xml下创建file_paths.xml，”黛琳说道。

```xml
<?xml version="1.0" encoding="utf-8"?>
<paths>
    <!-- 应用专属文件目录 -->
    <files-path name="my_files" path="files/" />
    
    <!-- 缓存目录 -->
    <cache-path name="my_cache" path="/" />
    
    <!-- 外部文件目录 -->
    <external-files-path name="external_files" path="." />
    
    <!-- 外部缓存 -->
    <external-cache-path name="external_cache" path="/" />
</paths>
```

## 1.9.3.4 分享文件

“现在可以分享文件了，”黛琳总结道。

```kotlin
fun shareFile(context: Context, file: File) {
    // 获取文件的content URI
    val uri = FileProvider.getUriForFile(
        context,
        "${context.packageName}.fileprovider",
        file
    )
    
    // 创建分享Intent
    val intent = Intent(Intent.ACTION_SEND).apply {
        type = context.getMimeType(file.absolutePath)
        putExtra(Intent.EXTRA_STREAM, uri)
        addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION)
    }
    
    context.startActivity(Intent.createChooser(intent, "分享文件"))
}

private fun getMimeType(path: String): String {
    val extension = MimeTypeMap.getFileExtensionFromUrl(path)
    return MimeTypeMap.getSingleton().getMimeTypeFromExtension(extension) 
        ?: "application/octet-stream"
}
```

## 1.9.3.5 专业技术总结

本章我们学习了如何共享文件。

**核心要点：**

1. **FileProvider生成安全URI** - 替代直接文件URI
2. **Manifest配置** - 声明Provider
3. **配置路径** - file_paths.xml
4. **添加权限flag** - FLAG_GRANT_READ_URI_PERMISSION
5. **处理MIME类型** - 正确设置文件类型

**路径配置：**

| 标签 | 目录 |
|------|------|
| files-path | filesDir |
| cache-path | cacheDir |
| external-files-path | getExternalFilesDir() |
| external-cache-path | externalCacheDir |
