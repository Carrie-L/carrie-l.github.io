# 1.9.4 请求其他App的文件

“如果我想从别的App获取文件，该怎么做？”洛FR问道。

“这和发送相反，用ACTION_GET_CONTENT这个Intent，”黛琳说道。

## 1.9.4.1 使用ACTION_GET_CONTENT

```kotlin
// 请求选择文件
fun requestFile() {
    val intent = Intent(Intent.ACTION_GET_CONTENT).apply {
        type = "*/*"  // 任意文件
        addCategory(Intent.CATEGORY_OPENABLE)
    }
    startActivityForResult(intent, REQUEST_CODE)
}

// 处理结果
override fun onActivityResult(requestCode: Int, resultCode: Int, data: Intent?) {
    if (requestCode == REQUEST_CODE && resultCode == RESULT_OK) {
        data?.data?.let { uri ->
            // 读取文件
            readFile(uri)
        }
    }
}

}
```

## 1.9.4.2 限制文件类型

“可以限制只能选择特定类型，”黛琳补充道。

```kotlin
// 只选择图片
val imageIntent = Intent(Intent.ACTION_GET_CONTENT).apply {
    type = "image/*"
    addCategory(Intent.CATEGORY_OPENABLE)
}

// 只选择PDF
val pdfIntent = Intent(Intent.ACTION_GET_CONTENT).apply {
    type = "application/pdf"
    addCategory(Intent.CATEGORY_OPENABLE)
}

// 只选择音频
val audioIntent = Intent(Intent.ACTION_GET_CONTENT).apply {
    type = "audio/*"
    addCategory(Intent.CATEGORY_OPENABLE)
}
```

## 1.9.4.3 获取持久权限

“如果要长期访问，需要获取持久权限，”黛琳强调道。

```kotlin
private fun takePersistablePermission(uri: Uri) {
    try {
        val takeFlags = Intent.FLAG_GRANT_READ_URI_PERMISSION or
                      Intent.FLAG_GRANT_WRITE_URI_PERMISSION
        contentResolver.takePersistableUriPermission(uri, takeFlags)
    } catch (e: SecurityException) {
        Log.e("Permission", "Failed to get permission", e)
    }
}

// 使用时检查权限
private fun hasPermission(uri: Uri): Boolean {
    val persistedUris = contentResolver.persistedUriPermissions
    return persistedUris.any { it.uri == uri && it.isReadPermission }
}
```

## 1.9.4.4 专业技术总结

本章我们学习了如何请求其他App共享的文件。

**核心要点：**

1. **使用ACTION_GET_CONTENT** - 让用户选择文件
2. **限制MIME类型** - image/*、application/pdf等
3. **读取文件** - openInputStream
4. **持久权限** - takePersistableUriPermission
5. **检查权限** - persistedUriPermissions

**常见MIME类型：**

| 类型 | MIME |
|------|------|
| 图片 | image/* |
| 音频 | audio/* |
| 视频 | video/* |
| PDF | application/pdf |
| 任意 | */* |
