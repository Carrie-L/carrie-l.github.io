# 20.1.1 命令行工具

"开发Android App会用到很多命令行工具，"黛琳介绍道。

## 常用命令位置

```bash
# Android SDK位置
$ANDROID_HOME/cmdline-tools/latest/bin/
# 或
$ANDROID_HOME/tools/bin/
```

## 常用工具

| 工具 | 用途 |
|------|------|
| adb | 调试桥 |
| aapt2 | 资源编译 |
| apksigner | APK签名 |
| zipalign | APK优化 |

## 环境变量

```bash
export ANDROID_HOME=/path/to/android/sdk
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin
```
