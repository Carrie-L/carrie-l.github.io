# 20.1.14 Logcat

"logcat查看日志是调试基础，"黛琳强调道。

## 基本用法

```bash
# 查看日志
adb logcat

# 过滤TAG
adb logcat -s MyTag

# 过滤级别
adb logcat *:E  # 只显示Error

# 清空日志
adb logcat -c
```

## 过滤表达式

```bash
# 过滤关键字
adb logcat | grep "keyword"

# 过滤标签和级别
adb logcat ActivityManager:I *:S
```
