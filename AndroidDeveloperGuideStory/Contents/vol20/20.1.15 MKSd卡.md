# 20.1.15 mksdcard

"mksdcard创建SD卡镜像文件，"黛琳介绍道。

```bash
# 创建SD卡镜像
mksdcard 512M sdcard.img

# 在模拟器中使用
emulator -sdcard sdcard.img
```
