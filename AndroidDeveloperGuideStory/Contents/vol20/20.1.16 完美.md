# 20.1.16 ProGuard

"ProGuard混淆和压缩代码，"黛琳介绍道。

## 配置

```groovy
# build.gradle
android {
    buildTypes {
        release {
            minifyEnabled true
            shrinkResources true
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
        }
    }
}
```

## 规则

```proguard
# 保持类
-keep class com.example.** { *; }

# 移除日志
-assumenosideeffects class android.util.Log {
    public static *** d(...);
}
```
