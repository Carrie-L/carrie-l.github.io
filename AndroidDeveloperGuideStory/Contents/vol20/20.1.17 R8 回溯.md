# 20.1.17 R8

"R8是ProGuard的继任者，"黛琳介绍道。

## 特点

- 比ProGuard更快
- 更好的混淆效果
- 支持更多的优化

## 配置

```groovy
android {
    buildTypes {
        release {
            minifyEnabled true
            shrinkResources true
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt')
        }
    }
}
```
