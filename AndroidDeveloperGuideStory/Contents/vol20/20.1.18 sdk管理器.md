# 20.1.18 sdkmanager

"sdkmanager管理SDK组件，"黛琳介绍道。

## 列出已安装

```bash
# 列出已安装包
sdkmanager --list_installed
```

## 安装组件

```bash
# 安装平台
sdkmanager "platforms;android-34"

# 安装构建工具
sdkmanager "build-tools;34.0.0"
```
