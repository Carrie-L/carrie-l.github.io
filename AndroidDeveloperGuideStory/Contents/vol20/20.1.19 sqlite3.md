# 20.1.19 sqlite3

"sqlite3操作SQLite数据库，"黛琳介绍道。

## 连接数据库

```bash
# 打开数据库
adb shell
sqlite3 /data/data/com.example/databases/app.db
```

## 基本操作

```sql
-- 查看表
.tables

-- 查看表结构
.schema users

-- 查询
SELECT * FROM users;

-- 插入
INSERT INTO users (name) VALUES ('test');

-- 退出
.exit
```
