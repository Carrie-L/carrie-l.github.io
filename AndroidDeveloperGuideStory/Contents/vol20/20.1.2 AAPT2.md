# 20.1.2 AAPT2

"AAPT2是Android资源编译器，"黛琳介绍道。

## 编译资源

```bash
# 编译XML
aapt2 compile path/to/resources -o output/

# 编译并优化
aapt2 compile --pseudo-localize -o compiled/
```

## 链接资源

```bash
# 生成APK
aapt2 link -o output.apk \
    -I android.jar \
    compiled/*.flat \
    AndroidManifest.xml
```
