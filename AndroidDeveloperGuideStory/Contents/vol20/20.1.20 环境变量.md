# 20.1.20 环境变量

"环境变量配置开发环境，"黛琳介绍道。

## 常用变量

| 变量 | 说明 |
|------|------|
| ANDROID_HOME | SDK根目录 |
| ANDROID_SDK_ROOT | SDK根目录(同ANDROID_HOME) |
| PATH | 添加命令行工具 |

## 配置

```bash
# ~/.bashrc 或 ~/.zshrc
export ANDROID_HOME=/usr/local/android-sdk
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools
```
