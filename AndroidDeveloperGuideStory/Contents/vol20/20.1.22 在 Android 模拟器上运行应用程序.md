# 20.1.22 模拟器

"模拟器是测试App的重要工具，"黛琳介绍道。

## 启动

```bash
# 启动模拟器
emulator -avd my_avd

# 带参数启动
emulator -avd my_avd -wipe-data -no-snapshot
```

## 命令行操作

```bash
# 列出设备
emulator -list-avds

# 关闭
adb emu kill
```
