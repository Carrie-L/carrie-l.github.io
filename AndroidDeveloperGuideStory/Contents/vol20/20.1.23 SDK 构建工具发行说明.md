# 20.1.23 SDK构建工具

"Build Tools包含编译APK的工具，"黛琳介绍道。

## 版本

常用版本：30.0.3, 31.0.0, 33.0.1, 34.0.0

## 包含工具

| 工具 | 用途 |
|------|------|
| aapt/aapt2 | 资源编译 |
| dx/d8 | DEX编译 |
| zipalign | APK对齐 |
| apksigner | APK签名 |
