# 20.1.3 ADB详解

"ADB是最重要的调试工具，"黛琳强调道。

## 连接设备

```bash
# 查看设备
adb devices

# 无线连接
adb connect 192.168.1.100:5555

# USB调试
adb usb
```

## 安装卸载

```bash
# 安装APK
adb install app.apk
adb install -r app.apk  # 覆盖安装

# 卸载
adb uninstall com.example.app
```

## 文件操作

```bash
# 推送文件到设备
adb push local.txt /sdcard/

# 拉取文件
adb pull /sdcard/file.txt ./

# 查看文件
adb shell ls -la
```

## 调试命令

```bash
# 查看日志
adb logcat

# 安装APK后自动启动
adb shell am start -n com.example/.MainActivity

# 强制停止应用
adb shell am force-stop com.example

# 查看应用信息
adb shell dumpsys package com.example
```
