# 20.1.4 apkanalyzer

"apkanalyzer分析APK组成，"黛琳介绍道。

## 基本用法

```bash
# 分析APK文件
apkanalyzer manifest print app.apk

# 查看文件大小
apkanalyzer resources --resources app.apk

# 查看DEX信息
apkanalyzer dex list app.apk

# 完整报告
apkanalyzer apks summarize app.apk
```
