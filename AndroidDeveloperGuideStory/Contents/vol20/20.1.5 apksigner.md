# 20.1.5 apksigner

"apksigner给APK签名，"黛琳介绍道。

## 签名APK

```bash
# 使用密钥签名
apksigner sign --ks mykey.jks --out signed.apk unsigned.apk

# 使用多个密钥
apksigner sign \
    --ks first-key.jks \
    --ks second-key.jks \
    --out signed.apk unsigned.apk
```

## 验证签名

```bash
# 验证签名
apksigner verify -v signed.apk

# 查看签名信息
apksigner verify --print-certs signed.apk
```
