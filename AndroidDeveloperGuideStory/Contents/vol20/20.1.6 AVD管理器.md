# 20.1.6 AVD管理器

"avdmanager管理Android虚拟设备，"黛琳介绍道。

## 列出设备

```bash
# 列出所有AVD
avdmanager list avd

# 列出系统镜像
avdmanager list sys
```

## 创建AVD

```bash
# 创建虚拟设备
avdmanager create avd -n my_avd -k "system-images;android-34;default;x86_64"
```

## 管理AVD

```bash
# 删除AVD
avdmanager delete avd -n my_avd
```
