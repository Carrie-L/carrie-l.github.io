# 20.1.7 bundletool

"bundletool处理App Bundle，"黛琳介绍道。

## 构建APK

```bash
# 从Bundle生成APK
bundletool build-apks \
    --bundle=app.aab \
    --output=app.apks \
    --ks=key.jks \
    --ks-pass=pass:123456
```

## 提取APK

```bash
# 从APKS中提取特定ABI
bundletool extract-apks \
    --apks=app.apks \
    --output-dir=output/ \
    --device-spec=device.json
```
