# 20.1.8 d8

"d8将class编译为DEX，"黛琳介绍道。

## 编译

```bash
# 编译DEX
d8 input.jar --output output/

# 编译多DEX
d8 input.jar --output output/ --main-dex-list main-dex.txt
```
