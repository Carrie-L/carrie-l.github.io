## 1. 权限类型

“Android 权限分为两种。”小 Com 介绍道：

| 类型 | 说明 | 需要用户授权 |
|------|------|-------------|
| **普通权限** | 低风险权限 | ❌ 自动授予 |
| **危险权限** | 高风险权限（位置、相机、存储等） | ✅ 需要用户同意 |

**常见危险权限：**

| 权限组 | 权限 | 用途 |
|--------|------|------|
| 相机 | `CAMERA` | 拍照、录像 |
| 位置 | `ACCESS_FINE_LOCATION`、`ACCESS_COARSE_LOCATION` | 定位 |
| 存储 | `READ_EXTERNAL_STORAGE`、`WRITE_EXTERNAL_STORAGE` | 文件读写 |
| 麦克风 | `RECORD_AUDIO` | 录音 |
| 联系人 | `READ_CONTACTS` | 读取联系人 |
| 通知 | `POST_NOTIFICATIONS` | 发送通知 |

---

## 2. 添加权限到 Manifest

“首先在 AndroidManifest.xml 声明权限。”小 Com 写道：

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android">
    
    <!-- 相机权限 -->
    <uses-permission android:name="android.permission.CAMERA" />
    
    <!-- 位置权限 -->
    <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
    <uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
    
    <!-- 存储权限 -->
    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"
        android:maxSdkVersion="32" />
    <uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
    
    <!-- 通知权限 (Android 13+) -->
    <uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
    
</manifest>
```

## 4. 请求相机权限

“首先看相机权限。”小 Com 展示了：

```kotlin
@Composable
fun CameraPermissionExample() {
    val cameraPermissionState = rememberPermissionState(
        Manifest.permission.CAMERA
    )
    
    when {
        cameraPermissionState.status.isGranted -> {
            // 有权限，显示相机预览
            CameraPreview()
        }
        else -> {
            // 没权限，显示请求按钮
            Column(
                modifier = Modifier.fillMaxSize(),
                horizontalAlignment = Alignment.CenterHorizontally,
                verticalArrangement = Arrangement.Center
            ) {
                Text("需要相机权限才能拍照")
                Spacer(modifier = Modifier.height(16.dp))
                Button(onClick = { cameraPermissionState.launchPermissionRequest() }) {
                    Text("授予权限")
                }
            }
        }
    }
}
```

## 5. 请求位置权限

“然后看位置权限。”小 Com 展示了：

```kotlin
@Composable
fun LocationPermissionExample() {
    val locationPermissionState = rememberMultiplePermissionsState(
        permissions = listOf(
            Manifest.permission.ACCESS_FINE_LOCATION,
            Manifest.permission.ACCESS_COARSE_LOCATION
        )
    )
    
    when {
        locationPermissionState.allPermissionsGranted -> {
            // 有权限，显示位置
            LocationContent()
        }
        locationPermissionState.shouldShowRationale -> {
            // 用户拒绝过，显示解释
            Column(
                modifier = Modifier.fillMaxSize(),
                horizontalAlignment = Alignment.CenterHorizontally,
                verticalArrangement = Arrangement.Center
            ) {
                Text("需要位置权限来获取您的位置信息")
                Spacer(modifier = Modifier.height(16.dp))
                Button(onClick = { locationPermissionState.launchMultiplePermissionRequest() }) {
                    Text("授予权限")
                }
            }
        }
        else -> {
            // 第一次请求
            Column(
                modifier = Modifier.fillMaxSize(),
                horizontalAlignment = Alignment.CenterHorizontally,
                verticalArrangement = Arrangement.Center
            ) {
                Text("我们需要您的位置权限")
                Spacer(modifier = Modifier.height(16.dp))
                Button(onClick = { locationPermissionState.launchMultiplePermissionRequest() }) {
                    Text("授予权限")
                }
            }
        }
    }
}
```

## 6. 请求通知权限（Android 13+）

“然后看通知权限。”小 Com 展示了：

```kotlin
@Composable
fun NotificationPermissionExample() {
    val notificationPermissionState = rememberPermissionState(
        Manifest.permission.POST_NOTIFICATIONS
    )
    
    // 检查系统版本
    val isAndroid13OrAbove = Build.VERSION.SDK_INT >= Build.VERSION_CODES.TIRAMISU
    
    when {
        !isAndroid13OrAbove -> {
            // Android 13 以下不需要请求
            NotificationContent()
        }
        notificationPermissionState.status.isGranted -> {
            // 有权限，发送通知
            NotificationContent()
        }
        else -> {
            // 没权限，显示请求
            Column {
                Text("需要通知权限来推送消息")
                Button(onClick = { notificationPermissionState.launchPermissionRequest() }) {
                    Text("授予权限")
                }
            }
        }
    }
}
```

## 7. 请求存储权限

“存储权限比较特殊，Android 13+ 改用媒体权限。”小 Com 展示了：

```kotlin
@Composable
fun StoragePermissionExample() {
    val storagePermissionState = rememberMultiplePermissionsState(
        permissions = if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.TIRAMISU) {
            listOf(Manifest.permission.READ_MEDIA_IMAGES)
        } else {
            listOf(Manifest.permission.READ_EXTERNAL_STORAGE)
        }
    )
    
    when {
        storagePermissionState.allPermissionsGranted -> {
            // 有权限，选择图片
            ImagePicker()
        }
        else -> {
            // 没权限
            Button(onClick = { storagePermissionState.launchMultiplePermissionRequest() }) {
                Text("授予存储权限")
            }
        }
    }
}
```

## 8. 一次性请求多个权限

“也可以一次性请求多个权限。”小 Com 展示了：

```kotlin
@Composable
fun MultiplePermissionsExample() {
    val permissionsState = rememberMultiplePermissionsState(
        permissions = listOf(
            Manifest.permission.CAMERA,
            Manifest.permission.ACCESS_FINE_LOCATION,
            Manifest.permission.RECORD_AUDIO
        )
    )
    
    Column(
        modifier = Modifier.fillMaxSize(),
        horizontalAlignment = Alignment.CenterHorizontally,
        verticalArrangement = Arrangement.Center
    ) {
        when {
            permissionsState.allPermissionsGranted -> {
                Text("所有权限都已授予！")
            }
            permissionsState.permissions.any { it.permission == Manifest.permission.CAMERA && it.status.isGranted } -> {
                Text("相机权限已授予")
            }
        }
        
        Spacer(modifier = Modifier.height(16.dp))
        
        Button(
            onClick = { permissionsState.launchMultiplePermissionRequest() }
        ) {
            Text("请求权限")
        }
    }
}
```

## 9. 实战：完整的权限请求页面

“我们来做最后一个练习——完整的权限请求页面！”小 Com 提议道。

```kotlin
@Composable
fun PermissionRequestScreen(
    onPermissionsGranted: () -> Unit
) {
    val cameraPermission = rememberPermissionState(Manifest.permission.CAMERA)
    val locationPermission = rememberPermissionState(Manifest.permission.ACCESS_FINE_LOCATION)
    val notificationPermission = rememberPermissionState(Manifest.permission.POST_NOTIFICATIONS)
    
    val allGranted = cameraPermission.status.isGranted &&
            locationPermission.status.isGranted &&
            notificationPermission.status.isGranted
    
    LaunchedEffect(allGranted) {
        if (allGranted) {
            onPermissionsGranted()
        }
    }
    
    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(24.dp),
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        Spacer(modifier = Modifier.height(48.dp))
        
        Icon(
            Icons.Default.Security,
            contentDescription = null,
            modifier = Modifier.size(80.dp),
            tint = MaterialTheme.colorScheme.primary
        )
        
        Spacer(modifier = Modifier.height(24.dp))
        
        Text(
            text = "需要一些权限",
            style = MaterialTheme.typography.headlineMedium
        )
        
        Spacer(modifier = Modifier.height(8.dp))
        
        Text(
            text = "为了提供更好的服务，需要以下权限：",
            style = MaterialTheme.typography.bodyMedium,
            color = Color.Gray
        )
        
        Spacer(modifier = Modifier.height(32.dp))
        
        // 权限列表
        PermissionItem(
            icon = Icons.Default.CameraAlt,
            title = "相机权限",
            description = "用于拍照和录像",
            isGranted = cameraPermission.status.isGranted,
            onRequest = { cameraPermission.launchPermissionRequest() }
        )
        
        PermissionItem(
            icon = Icons.Default.LocationOn,
            title = "位置权限",
            description = "用于获取您的位置",
            isGranted = locationPermission.status.isGranted,
            onRequest = { locationPermission.launchPermissionRequest() }
        )
        
        PermissionItem(
            icon = Icons.Default.Notifications,
            title = "通知权限",
            description = "用于推送消息",
            isGranted = notificationPermission.status.isGranted,
            onRequest = { notificationPermission.launchPermissionRequest() }
        )
        
        Spacer(modifier = Modifier.weight(1f))
        
        if (allGranted) {
            Text(
                text = "所有权限已授予！",
                color = Color.Green
            )
        }
    }
}

@Composable
fun PermissionItem(
    icon: ImageVector,
    title: String,
    description: String,
    isGranted: Boolean,
    onRequest: () -> Unit
) {
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .padding(vertical = 8.dp)
    ) {
        Row(
            modifier = Modifier
                .fillMaxWidth()
                .padding(16.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            Icon(
                icon,
                contentDescription = null,
                tint = if (isGranted) Color.Green else Color.Gray
            )
            
            Spacer(modifier = Modifier.width(16.dp))
            
            Column(modifier = Modifier.weight(1f)) {
                Text(title, style = MaterialTheme.typography.titleSmall)
                Text(
                    description,
                    style = MaterialTheme.typography.bodySmall,
                    color = Color.Gray
                )
            }
            
            if (isGranted) {
                Icon(
                    Icons.Default.Check,
                    contentDescription = null,
                    tint = Color.Green
                )
            } else {
                TextButton(onClick = onRequest) {
                    Text("授权")
                }
            }
        }
    }
}
```
