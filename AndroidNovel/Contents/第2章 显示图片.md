# 第二课：显示图片 · Image 组件详解

---

“叮——”

又是那个熟悉的声音。林小满猛地睁开眼睛，发现自己已经站在了一片茂密的森林里。阳光穿过树叶的缝隙，在地面上洒下斑驳的光影。

“小满！这边！”

远处，小 Com 正在一棵大树下向她招手。在她身后，是一座精巧的小木屋，屋顶上还有一个风车在慢慢转动。

“昨天睡得好吗？”小 Com 笑眯眯地问。

“还……还行。”林小满看了看自己身上的白色长袍，“就是醒来的时候有点懵。”

“习惯就好啦！”小 Com 拉起她的手，“今天我们要学的东西可有趣了——怎么在 Compose 里显示图片！”

“图片？”林小满眼睛一亮，“是不是就像手机相册里的照片那样？”

“没错！不过在 Compose 里，显示图片可比以前方便多了！”小 Com 得意地说。

## 走进图片小屋

她们来到那座小木屋前。小 Com 推开门，里面是一个温馨的小房间，墙上挂满了各种画作。

“这些都是用 Compose 画的？”林小满惊讶地问。

“对！每一幅画都是一个 Image 组件。”小 Com 解释道，“在 Compose 里，显示图片非常非常简单，只需要一个 `Image` 函数就够了。”

她在桌子上放了一台笔记本电脑：“我们来看代码。”

---

## 第一个 Image

小 Com 打开 Android Studio，新建了一个 Compose 项目，然后写下了这段代码：

```kotlin
Image(
    painter = painterResource(R.drawable.sample),
    contentDescription = "示例图片"
)
```

“就……这么短？”林小满不敢相信自己的眼睛。

“就这么短！”小 Com 笑着说，“`painterResource(R.drawable.sample)` 会自动加载 `res/drawable` 目录下的图片资源。`contentDescription` 是给屏幕阅读器用的Accessibility 内容描述。”

---

## 加载网络图片

“那……如果是网上的图片呢？”林小满问。

小 Com 打了个响指：“问得好！网上的图片，需要用 Coil 这个库！”

她继续写道：

```kotlin
// 需要先添加依赖
// implementation("io.coil-kt:coil-compose:2.5.0")

Image(
    painter = rememberAsyncImagePainter("https://example.com/photo.jpg"),
    contentDescription = "网络图片"
)
```

“Coil 是 Compose 生态里最流行的图片加载库，”小 Com 解释道，“它会自动处理图片的下载、缓存、占位图……你完全不用操心。”

“这么方便！”林小满感叹道。

---

## 图片的样式：contentScale

显示图片还需要控制它的大小、填充方式。小 Com 又展示了几个参数：

```kotlin
Image(
    painter = painterResource(R.drawable.landscape),
    contentDescription = "风景图",
    contentScale = ContentScale.Crop,  // 剪裁填充
    modifier = Modifier
        .size(200.dp)
        .clip(CircleShape)  // 圆形剪裁
)
```

小 Com 列举了几种 `ContentScale` 模式：

| 模式 | 效果 |
|------|------|
| `FillBounds` | 拉伸填满（会变形） |
| `Fit` | 适应容器（保持比例） |
| `Crop` | 剪裁填充（常用） |
| `Inside` | 缩放适应（留白） |

---

## Modifier：图片的“装修师”

“在 Compose 里，所有的视觉效果都可以用 `Modifier` 来调整。”小 Com 说，“图片也不例外。”

她列举了一些常用的 Modifier：

```kotlin
Image(
    painter = painterResource(R.drawable.photo),
    contentDescription = "照片",
    modifier = Modifier
        .size(150.dp)           // 大小
        .clip(RoundedCornerShape(16.dp))  // 圆角
        .border(2.dp, Color.Gray, RoundedCornerShape(16.dp))  // 边框
        .shadow(4.dp, RoundedCornerShape(16.dp))  // 阴影
)
```

“哇！”林小满眼睛闪闪发光，“这比以前的 XML 简单太多了！”

“那是！”小 Com 骄傲地说，“在 Compose 里，一切都变成 Kotlin 代码，想怎么改就怎么改。”

## 实战：做一个图片卡片

“好，我们来做一个小练习！”小 Com 提议道。

她设计了这样一个卡片：

```kotlin
Card(
    modifier = Modifier
        .fillMaxWidth()
        .padding(16.dp),
    shape = RoundedCornerShape(16.dp),
    elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
) {
    Column {
        Image(
            painter = painterResource(R.drawable.card_image),
            contentDescription = "卡片图片",
            contentScale = ContentScale.Crop,
            modifier = Modifier
                .fillMaxWidth()
                .height(200.dp)
        )
        
        Text(
            text = "美丽的风景",
            style = MaterialTheme.typography.titleMedium,
            modifier = Modifier.padding(16.dp)
        )
    }
}
```

“这就是一个完整的图片卡片！”小 Com 解释道，“Card 是 Material Design 的卡片组件，里面放了一个 Image 和一个 Text。”

林小满跟着写了一遍，果真成功了！

## 本课小结

这一天，林小满学到了：

1. **Image 组件**：Compose 里显示图片的核心组件
2. **painterResource**：加载本地图片资源
3. **Coil 库**：加载网络图片的首选方案
4. **contentScale**：控制图片的填充方式
5. **Modifier**：调整图片的大小、圆角、边框、阴影
