# 第三课：按钮与点击事件 · 让界面动起来

---

“叮——”

林小满发现自己站在一个热闹的广场上。广场中央有一个大大的舞台，周围摆满了各种小吃摊。

“今天是什么日子？”她好奇地问。

小 Com 从身后拍了拍她的肩膀：“今天是 Compose 村的互动日！所有人都要学会做按钮，这样大家才能互相交流！”

“按钮？”林小满看了看四周，果然看到每个人都拿着一个手机，屏幕上全是各种按钮。

“对！”小 Com 说，“没有按钮，就没有办法交互。在 Compose 里，按钮是最重要的交互组件。”

---

## 第一个按钮

小 Com 带着林小满来到舞台边的摊位，那里有一台笔记本电脑。

“首先，我们来看最基本的按钮——`Button`。”小 Com 写道：

```kotlin
Button(
    onClick = { /* 点击时要做什么 */ }
) {
    Text("点击我")
}
```

“`onClick` 是一个 lambda 函数，”小 Com 解释道，“当用户点击按钮时，就会执行里面的代码。”

林小满问：“那我想点击后显示一个提示，该怎么做？”

“简单！”小 Com 展示了完整的例子：

```kotlin
var showMessage by remember { mutableStateOf(false) }

Button(
    onClick = { showMessage = true }
) {
    Text("点击我")
}

if (showMessage) {
    Text("你点到了！")
}
```

---

## 按钮的种类

“按钮有很多种，”小 Com 介绍道，“不同场景用不同的按钮。”

| 按钮类型 | 用途 | 代码 |
|----------|------|------|
| `Button` | 主要操作 | 主色调，实心 |
| `OutlinedButton` | 次要操作 | 空心边框 |
| `TextButton` | 文字链接 | 只有文字 |
| `IconButton` | 图标按钮 | 只有图标 |

小 Com 逐一演示：

```kotlin
// 主要按钮
Button(onClick = { }) { Text("确认") }

// 次要按钮
OutlinedButton(onClick = { }) { Text("取消") }

// 文字按钮
TextButton(onClick = { }) { Text("了解更多") }

// 图标按钮
IconButton(onClick = { }) {
    Icon(Icons.Default.Favorite, contentDescription = "收藏")
}
```

---

## 带图标的按钮

“我想让按钮上面有个图标，该怎么做？”林小满问。

小 Com 笑道：“很简单，把 Icon 放在 Button 或者 Text 里面就行了！”

```kotlin
Button(
    onClick = { /* ... */ },
    leadingIcon = {  // 图标在文字左边
        Icon(
            Icons.Default.Send,
            contentDescription = null
        )
    }
) {
    Text("发送")
}

// 或者图标在右边
Button(
    onClick = { /* ... */ },
    trailingIcon = {  // 图标在文字右边
        Icon(
            Icons.Default.ArrowForward,
            contentDescription = null
        )
    }
) {
    Text("下一步")
}
```

## 按钮的 enabled 状态

“有的时候，按钮是不可点击的，比如表单没填完的时候。”小 Com 说。

```kotlin
var isEnabled by remember { mutableStateOf(false) }

Button(
    onClick = { /* ... */ },
    enabled = isEnabled  // 控制是否可点击
) {
    Text("提交")
}
```

“当 `enabled = false` 时，按钮会自动变灰，无法点击。”小 Com 解释道。

## 实战：做一个计数器

“我们来做个好玩的！”小 Com 提议道，“做一个计数器应用！”

```kotlin
var count by remember { mutableStateOf(0) }

Column(
    horizontalAlignment = Alignment.CenterHorizontally,
    modifier = Modifier.padding(32.dp)
) {
    Text(
        text = "计数：$count",
        style = MaterialTheme.typography.headlineLarge
    )
    
    Spacer(modifier = Modifier.height(16.dp))
    
    Row(
        horizontalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        // 减少按钮
        OutlinedButton(
            onClick = { count-- }
        ) {
            Icon(Icons.Default.Remove, contentDescription = "减少")
            Text(" -1")
        }
        
        // 增加按钮
        Button(
            onClick = { count++ }
        ) {
            Icon(Icons.Default.Add, contentDescription = "增加")
            Text(" +1")
        }
    }
    
    Spacer(modifier = Modifier.height(16.dp))
    
    // 重置按钮
    TextButton(
        onClick = { count = 0 }
    ) {
        Text("重置")
    }
}
```

“太棒了！”林小满兴奋地说，“这就是一个完整的计数器应用！”

## 更多交互组件

“在 Compose 里，还有很多交互组件，原理都差不多。”小 Com 介绍道：

```kotlin
// 复选框
Checkbox(
    checked = checked,
    onCheckedChange = { checked = it }
)

// 开关
Switch(
    checked = checked,
    onCheckedChange = { checked = it }
)

// 单选框
RadioButton(
    selected = selected,
    onClick = { selected = true }
)

// 滑块
Slider(
    value = value,
    onValueChange = { value = it }
)
```

“所有的交互组件，都遵循同样的模式：`状态 + onXxxChange`。”小 Com 总结道。
