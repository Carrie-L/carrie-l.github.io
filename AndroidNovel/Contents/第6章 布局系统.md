# 第六课：布局系统 · Row / Column / Box

---

“叮——”

林小满发现自己站在一片工地旁边。远处有吊车在运转，近处堆满了各种建筑材料。

“今天我们要在 Compose 村里盖房子！”小 Com 戴着安全帽，手里拿着图纸走了过来。

“盖……房子？”林小满一脸问号。

“对！”小 Com 笑着说，“在 Compose 里，构建界面就像盖房子一样。你需要不同的布局组件来'摆放'各种元素——这就是我们今天要学的：Row、Column、Box！”

---

## 布局三兄弟

小 Com 指向远处的三座建筑：

“在 Compose 里，最常用的布局组件有三个，我叫它们'布局三兄弟'：”

| 组件 | 方向 | 比喻 |
|------|------|------|
| `Row` | 水平 | 一排房子 |
| `Column` | 垂直 | 一栋楼 |
| `Box` | 叠放 | 堆积木 |

---

## Row：水平布局

“首先看 `Row`——水平布局。”小 Com 在电脑上画图：

```kotlin
Row(
    modifier = Modifier.fillMaxWidth(),
    horizontalArrangement = Arrangement.SpaceBetween
) {
    Text("左边")
    Text("中间")
    Text("右边")
}
```

“`horizontalArrangement` 控制水平对齐方式：”

| 方式 | 效果 |
|------|------|
| `Start` | 左对齐（默认） |
| `CenterHorizontally` | 居中 |
| `End` | 右对齐 |
| `SpaceBetween` | 等距分布 |
| `SpaceAround` | 间距相等 |
| `SpaceEvenly` | 元素间和边缘间距相等 |

---

## Column：垂直布局

“然后是 `Column`——垂直布局。”小 Com 继续写：

```kotlin
Column(
    modifier = Modifier.fillMaxSize(),
    verticalArrangement = Arrangement.SpaceEvenly
) {
    Text("第一行")
    Text("第二行")
    Text("第三行")
}
```

“`verticalArrangement` 控制垂直对齐方式：”

| 方式 | 效果 |
|------|------|
| `Top` | 顶部对齐（默认） |
| `CenterVertically` | 垂直居中 |
| `Bottom` | 底部对齐 |
| `SpaceBetween` | 等距分布 |
| `SpaceAround` | 间距相等 |
| `SpaceEvenly` | 元素间和边缘间距相等 |

---

## Box：层叠布局

“最后是 `Box`——层叠布局。”小 Com 写道：

```kotlin
Box(
    modifier = Modifier
        .size(200.dp)
        .background(Color.Blue)
) {
    Text(
        "在底部",
        modifier = Modifier.align(Alignment.BottomCenter)
    )
    Text(
        "在中间",
        modifier = Modifier.align(Alignment.Center)
    )
    Text(
        "在顶部",
        modifier = Modifier.align(Alignment.TopCenter)
    )
}
```

“`align` 可以控制元素在 Box 内的位置。”小 Com 解释。

Box 的对齐方式有：

| 方向 | 选项 |
|------|------|
| 水平 | `Start`, `CenterHorizontally`, `End` |
| 垂直 | `Top`, `CenterVertically`, `Bottom` |
| 全部 | `Center`, `TopStart`, `TopEnd`, 等 |

## Modifier：布局的“装修师”

“除了布局组件，我们还需要用 `Modifier` 来调整元素的位置和大小。”小 Com 说。

### padding：内边距

```kotlin
Text(
    "有内边距的文字",
    modifier = Modifier.padding(16.dp)
)

// 也可以单独设置某一侧
modifier = Modifier.padding(start = 8.dp, top = 16.dp)
```

### margin：外边距

“Compose 里没有单独的 margin，通常用 Spacer 代替：”

```kotlin
Row {
    Text("元素1")
    Spacer(modifier = Modifier.width(16.dp))  // 水平间距
    Text("元素2")
}

Column {
    Text("元素1")
    Spacer(modifier = Modifier.height(16.dp))  // 垂直间距
    Text("元素2")
}
```

### size：大小

```kotlin
// 固定大小
Modifier.size(100.dp)

// 宽度固定，高度自适应
Modifier.fillMaxWidth()

// 高度固定，宽度自适应
Modifier.fillMaxHeight()

// 完全填充父容器
Modifier.fillMaxSize()
```

## 实战：做一个个人资料卡片

“我们来做一个小项目——个人资料卡片！”小 Com 提议道。

```kotlin
Card(
    modifier = Modifier
        .fillMaxWidth()
        .padding(16.dp),
    shape = RoundedCornerShape(16.dp),
    elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
) {
    Column(
        modifier = Modifier.padding(16.dp),
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        // 头像
        Image(
            painter = painterResource(R.drawable.avatar),
            contentDescription = "头像",
            modifier = Modifier
                .size(80.dp)
                .clip(CircleShape)
        )
        
        Spacer(modifier = Modifier.height(12.dp))
        
        // 名字
        Text(
            text = "林小满",
            style = MaterialTheme.typography.titleLarge
        )
        
        // 身份
        Text(
            text = "Android 开发者",
            style = MaterialTheme.typography.bodyMedium,
            color = Color.Gray
        )
        
        Spacer(modifier = Modifier.height(16.dp))
        
        // 统计数据
        Row(
            modifier = Modifier.fillMaxWidth(),
            horizontalArrangement = Arrangement.SpaceEvenly
        ) {
            StatItem(count = "128", label = "粉丝")
            StatItem(count = "256", label = "关注")
            StatItem(count = "512", label = "获赞")
        }
    }
}

@Composable
fun StatItem(count: String, label: String) {
    Column(horizontalAlignment = Alignment.CenterHorizontally) {
        Text(
            text = count,
            style = MaterialTheme.typography.titleMedium
        )
        Text(
            text = label,
            style = MaterialTheme.typography.bodySmall,
            color = Color.Gray
        )
    }
}
```

“太漂亮了！”林小满惊呼。

## 嵌套布局

“复杂的界面通常需要嵌套使用这些布局。”小 Com 展示了一个更复杂的例子：

```kotlin
Column(
    modifier = Modifier.fillMaxSize()
) {
    // 顶部标题栏
    Row(
        modifier = Modifier
            .fillMaxWidth()
            .background(Color.Blue)
            .padding(16.dp),
        horizontalArrangement = Arrangement.SpaceBetween,
        verticalAlignment = Alignment.CenterVertically
    ) {
        Text("标题", color = Color.White)
        Icon(Icons.Default.Settings, contentDescription = "设置", tint = Color.White)
    }
    
    // 中间内容区
    Box(
        modifier = Modifier
            .weight(1f)
            .fillMaxWidth()
    ) {
        // 这里放主要 content
    }
    
    // 底部导航栏
    Row(
        modifier = Modifier
            .fillMaxWidth()
            .background(Color.LightGray)
            .padding(8.dp),
        horizontalArrangement = Arrangement.SpaceEvenly
    ) {
        IconButton(onClick = { }) { Icon(Icons.Default.Home, "首页") }
        IconButton(onClick = { }) { Icon(Icons.Default.Search, "搜索") }
        IconButton(onClick = { }) { Icon(Icons.Default.Person, "我的") }
    }
}
```

## Spacer 的妙用

“在布局中，Spacer 用来创造空白间距，非常有用。”小 Com 总结了一些技巧：

```kotlin
// 固定高度的空白
Spacer(modifier = Modifier.height(16.dp))

// 固定宽度的空白
Spacer(modifier = Modifier.width(8.dp))

// 正方形的空白（用于 Grid）
Spacer(modifier = Modifier.size(16.dp))

// 可变大小的空白（用 weight）
Column {
    Text("顶部")
    Spacer(modifier = Modifier.weight(1f))  // 占据剩余空间
    Text("底部")
}
```
