# 第七课：修饰符大集 · background / border / shadow

---

“叮——”

林小满发现自己坐在一间美术教室里。墙上挂满了各种画作，桌上摆满了颜料和画笔。

“今天我们要当设计师！”小 Com 穿着围裙走了过来，“用修饰符（Modifier）来美化我们的界面！”

“修饰符？”林小满问。

“对！”小 Com 说，“修饰符就像画笔，可以给 UI 元素添加背景、边框、阴影、圆角……各种视觉效果！”

---

## 背景色：background

“首先是背景色——`background`。”小 Com 拿起画笔：

```kotlin
Box(
    modifier = Modifier
        .size(100.dp)
        .background(Color.Red)
)
```

“颜色可以是直接的：”

```kotlin
.background(Color.Red)      // 红色
.background(Color.Blue)     // 蓝色
.background(Color.Yellow)   // 黄色
.background(Color.Gray)    // 灰色
```

“也可以用十六进制：”

```kotlin
.background(Color(0xFF3498DB))  // 蓝色
.background(Color(0xFF2ECC71))  // 绿色
.background(Color(0xFFE74C3C))  // 红色

// 带透明度
.background(Color(0x80000000))  // 50% 黑色
```

---

## 渐变背景

“还可以做渐变背景！”小 Com 展示了更高级的用法：

```kotlin
// 线性渐变
Modifier.background(
    brush = Brush.linearGradient(
        colors = listOf(Color.Red, Color.Blue)
    )
)

// 水平渐变
Modifier.background(
    brush = Brush.horizontalGradient(
        colors = listOf(Color(0xFF3498DB), Color(0xFF9B59B6))
    )
)

// 垂直渐变
Modifier.background(
    brush = Brush.verticalGradient(
        colors = listOf(Color(0xFF3498DB), Color(0xFF2ECC71))
    )
)

// 径向渐变
Modifier.background(
    brush = Brush.radialGradient(
        colors = listOf(Color.Yellow, Color.Red)
    )
)
```

---

## 圆角：clip

“接下来是圆角——`clip`。”小 Com 写道：

```kotlin
// 圆角矩形
Modifier.clip(RoundedCornerShape(16.dp))

// 圆形（等于圆角为一半）
Modifier.clip(CircleShape)

// 切角（斜角）
Modifier.clip(CornerSize(50 percent))
```

“`clip` 需要和 `background` 配合使用：”

```kotlin
Modifier
    .background(Color.Blue, RoundedCornerShape(16.dp))
    // 或者用 clip
    .clip(RoundedCornerShape(16.dp))
    .background(Color.Blue)
```

---

## 边框：border

“然后是边框——`border`。”小 Com 展示了代码：

```kotlin
// 简单边框
Modifier.border(2.dp, Color.Red)

// 圆角边框
Modifier.border(2.dp, Color.Red, RoundedCornerShape(8.dp))

// 虚线边框
Modifier.border(
    width = 2.dp,
    color = Color.Red,
    shape = RoundedCornerShape(8.dp)
).drawWithContent {
    // 需要自定义实现虚线效果
}
```

“也可以用 Brush 做渐变边框：”

```kotlin
Modifier.border(
    width = 3.dp,
    brush = Brush.linearGradient(
        colors = listOf(Color.Red, Color.Blue, Color.Green)
    ),
    shape = RoundedCornerShape(16.dp)
)
```

---

## 阴影：shadow

“接下来是阴影——`shadow`。”小 Com 越来越兴奋：

```kotlin
// 基础阴影
Modifier.shadow(4.dp, RoundedCornerShape(8.dp))

// 大阴影
Modifier.shadow(8.dp, CircleShape)

// 多个阴影（用 layer）
Modifier
    .shadow(4.dp, RoundedCornerShape(8.dp))
    .background(Color.White)
```

“在 Material Design 3 里，更推荐用 `Card` 组件：”

```kotlin
Card(
    elevation = CardDefaults.cardElevation(
        defaultElevation = 4.dp
    )
) {
    Text("有阴影的卡片")
}
```

## 实战：做一个炫酷的按钮

“我们来做一个小练习——一个炫酷的按钮！”小 Com 提议道。

```kotlin
@Composable
fun CoolButton(
    text: String,
    onClick: () -> Unit
) {
    Box(
        modifier = Modifier
            .clip(RoundedCornerShape(24.dp))
            .background(
                brush = Brush.horizontalGradient(
                    colors = listOf(
                        Color(0xFF667EEA),
                        Color(0xFF764BA2)
                    )
                )
            )
            .shadow(8.dp, RoundedCornerShape(24.dp))
            .clickable(onClick = onClick)
            .padding(horizontal = 32.dp, vertical = 16.dp),
        contentAlignment = Alignment.Center
    ) {
        Text(
            text = text,
            color = Color.White,
            style = MaterialTheme.typography.titleMedium
        )
    }
}
```

“哇！好好看！”林小满说。

## 点击效果：clickable

“修饰符还可以添加点击效果！”小 Com 展示了代码：

```kotlin
// 简单点击
Modifier.clickable { /* 点击事件 */ }

// 带波纹效果（Material Design）
Modifier.clickable { /* ... */ }

// 带波纹 + 点击音效
Modifier
    .clickable(
        interactionSource = remember { MutableInteractionSource() },
        indication = rememberRipple()
    ) { /* ... */ }
```

## 实战：做一个可点击的卡片

```kotlin
@Composable
fun ClickableCard(
    title: String,
    description: String,
    onClick: () -> Unit
) {
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp)
            .clickable(onClick = onClick),
        shape = RoundedCornerShape(16.dp),
        elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
    ) {
        Column(modifier = Modifier.padding(16.dp)) {
            Text(
                text = title,
                style = MaterialTheme.typography.titleMedium
            )
            Spacer(modifier = Modifier.height(8.dp))
            Text(
                text = description,
                style = MaterialTheme.typography.bodyMedium,
                color = Color.Gray
            )
        }
    }
}
```

## 常用修饰符一览

小 Com 整理了一个表格：

| 修饰符 | 用途 | 示例 |
|--------|------|------|
| `size` | 大小 | `.size(100.dp)` |
| `fillMaxWidth` | 宽度填满 | `.fillMaxWidth()` |
| `padding` | 内边距 | `.padding(16.dp)` |
| `background` | 背景色 | `.background(Color.Red)` |
| `clip` | 裁剪 | `.clip(RoundedCornerShape(8.dp))` |
| `border` | 边框 | `.border(2.dp, Color.Red)` |
| `shadow` | 阴影 | `.shadow(4.dp)` |
| `clickable` | 可点击 | `.clickable { }` |
| `draggable` | 拖拽 | `.draggable { }` |
| `scrollable` | 滚动 | `.scrollable { }` |

## 实战：做一个完整的登录卡片

“我们来做最后一个练习——登录卡片！”小 Com 说。

```kotlin
@Composable
fun LoginCard() {
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .padding(24.dp),
        shape = RoundedCornerShape(24.dp),
        elevation = CardDefaults.cardElevation(defaultElevation = 8.dp)
    ) {
        Column(
            modifier = Modifier.padding(24.dp),
            horizontalAlignment = Alignment.CenterHorizontally
        ) {
            // 标题
            Text(
                text = "欢迎登录",
                style = MaterialTheme.typography.headlineSmall
            )
            
            Spacer(modifier = Modifier.height(24.dp))
            
            // 用户名输入框
            OutlinedTextField(
                value = "",
                onValueChange = { },
                label = { Text("用户名") },
                modifier = Modifier.fillMaxWidth(),
                shape = RoundedCornerShape(12.dp)
            )
            
            Spacer(modifier = Modifier.height(16.dp))
            
            // 密码输入框
            OutlinedTextField(
                value = "",
                onValueChange = { },
                label = { Text("密码") },
                modifier = Modifier.fillMaxWidth(),
                shape = RoundedCornerShape(12.dp),
                visualTransformation = PasswordVisualTransformation()
            )
            
            Spacer(modifier = Modifier.height(24.dp))
            
            // 登录按钮
            Button(
                onClick = { },
                modifier = Modifier
                    .fillMaxWidth()
                    .height(56.dp),
                shape = RoundedCornerShape(12.dp),
                colors = ButtonDefaults.buttonColors(
                    containerColor = Color(0xFF667EEA)
                )
            ) {
                Text("登录", style = MaterialTheme.typography.titleMedium)
            }
            
            Spacer(modifier = Modifier.height(16.dp))
            
            // 忘记密码
            TextButton(onClick = { }) {
                Text("忘记密码？", color = Color.Gray)
            }
        }
    }
}
```
