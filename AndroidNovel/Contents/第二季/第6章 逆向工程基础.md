## 4. APK 结构

“APK 里面有什么？”小 Com 展示了：

```
app.apk/
├── AndroidManifest.xml    # 应用清单（二进制）
├── classes.dex           # Dalvik 字节码
├── classes2.dex          # 第二个 dex
├── resources.arsc       # 资源索引
├── res/                 # 资源文件
│   ├── drawable/
│   ├── layout/
│   └── values/
├── assets/              # 原始资源
├── lib/                 # so 库
└── META-INF/           # 签名信息
```

## 6. 分析 Java 代码

“分析反编译的 Java 代码。”小 Com 展示了：

```kotlin
// 反编译后的代码
public class LoginActivity extends AppCompatActivity {
    // 反编译可能得到这样的代码：
    
    private void login(String username, String password) {
        if (TextUtils.isEmpty(username)) {
            Toast.makeText(this, "用户名不能为空", 0).show();
            return;
        }
        // 调用网络请求
        Retrofit retrofit = new Retrofit.Builder()
            .baseUrl("https://api.example.com/")
            .build();
        ApiService service = retrofit.create(ApiService.class);
        service.login(username, password).enqueue(new Callback<LoginResponse>() {
            public void onResponse(Call<LoginResponse> call, Response<LoginResponse> response) {
                // 登录成功
            }
            
            public void onFailure(Call<LoginResponse> call, Throwable t) {
                // 登录失败
            }
        });
    }
}
```

## 7. 动态分析：Frida

“用 Frida 做动态分析。”小 Com 展示了：

```python
# frida 脚本
import frida
import sys

# 附加到进程
session = frida.attach("com.example.app")

# Hook 方法
script = session.create_script("""
    // Hook login 方法
    Java.perform(function() {
        var LoginActivity = Java.use("com.example.app.LoginActivity");
        LoginActivity.login.implementation = function(username, password) {
            // 打印参数
            console.log("Username: " + username);
            console.log("Password: " + password);
            
            // 调用原始方法
            this.login(username, password);
        };
    });
""")

script.load()
sys.stdin.read()
```

## 9. 绕过 SSL Pinning

“有时需要绕过 SSL Pinning。”小 Com 展示了：

```javascript
// 绕过 SSL Pinning
Java.perform(function() {
    var CertificateFactory = Java.use("javax.net.ssl.CertificateFactory");
    var SSLContext = Java.use("javax.net.ssl.SSLContext");
    var TrustManager = Java.use("javax.net.ssl.X509TrustManager");
    
    // Hook OkHttp
    var OkHttpClient = Java.use("okhttp3.OkHttpClient");
    var Builder = Java.use("okhttp3.OkHttpClient$Builder");
    
    Builder.sslSocketFactory.implementation = function(sslSocketFactory, trustManager) {
        console.log("Bypassing SSL Pinning!");
        return this.sslSocketFactory(Java.use("javax.net.ssl.SSLSocketFactory").getDefault(), 
                                    Java.use("javax.net.ssl.X509TrustManager").$new());
    };
});
```
