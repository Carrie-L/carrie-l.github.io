## 用户档案
- **身份**：安卓开发工程师，空窗4年后重新入职

- **算法基础**：较弱，需要系统训练
- **英文水平**：刚过六级

## 学习策略
1. **按题型分组**：数组 → 字符串 → 链表 → 二叉树 → 动态规划 → 图论
2. **难度递增**：简单(1-2星) → 中等(3星) → 困难(4-5星)
3. **每周一个主题**：周二~周五新题型，周一复习+总结
4. **代码量要求**：每天至少2道题，周末复盘
