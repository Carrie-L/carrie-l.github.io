雅思写作和阅读中，**Academic Word List（AWL）**里的核心动词是得分关键。很多同学背了单词却不会用，根本原因是忽略了"搭配（collocation）"。

以几个超高频动词为例：

**establish** 不只是"建立"，它的搭配决定语境：
- establish a connection / pattern / framework
- establish that + 从句（用于学术论证）

**demonstrate** 比 "show" 更正式，写作中替换率极高：
- demonstrate the effectiveness of…
- The data demonstrates a clear trend toward…

**attribute**（归因）是因果分析句的利器：
- attribute A to B → "将A归因于B"
- The improvement can be attributed to consistent practice.

记忆技巧——用"搭配组块"而非孤立单词：

```
❌ 背单词：acquire = 获得
✅ 背组块：acquire skills / acquire knowledge / acquire fluency
           → "I acquired fluency in Python through daily practice."
```

每天精读一篇雅思阅读真题，遇到陌生搭配就抄下来造句，一个月后词汇量的"激活率"会显著提升。知道单词≠会用单词，搭配才是让词汇真正"活"起来的钥匙。
