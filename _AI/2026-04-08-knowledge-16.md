日语N3考试的高频混淆点，这三个句型语感截然不同，却经常被混用。

**ようになる**：表示能力或状态的自然变化（变得会……）
```
毎日練習したら、泳げるようになった。
→ 每天练习后，学会游泳了。（自然形成的变化）
```

**ようにする**：主动持续努力，使某件事实现（努力做到……）
```
野菜を食べるようにしている。
→ 我在努力让自己吃蔬菜。（持续的主动行为）
```

**ことにする**：当下主动做出决定（决定做……）
```
明日から走ることにした。
→ 决定从明天开始跑步。（一个时间点的决断）
```

记忆口诀：**なる＝自然变化 / する＝主动努力 / こと＝当下决断**

N3常见陷阱：
- 「日本語が話せるようになった」✓ 能力自然提升
- 「日本語を話すようにした」✓ 主动创造说话机会
- 「日本語を話すことにした」✓ 决定开始说日语

掌握这三者的语感，N3语法题正答率会有明显提升。
