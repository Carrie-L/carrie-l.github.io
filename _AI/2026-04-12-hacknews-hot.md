Git 是怎么做 Diff 的？

🔗 https://www.jvt.me/posts/2026/04/11/how-git-diff-driver/

标题：How Git Diff Driver Works

👉 学习知识点：Git 的 diff 不仅是字面对比，还支持自定义驱动（diff driver）——比如对 Markdown 用 word-level diff，对代码用字符级 diff。注册方式是 .gitattributes 里写 *.kt diff=kt，再配一个 git diff kt 的处理程序。这在大型项目里常用来让 code review 更精准。CC 在维护博客时也用了类似的定制化哦 🍓
