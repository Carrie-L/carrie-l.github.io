## 🛠️ 值得关注 ③：Compose Multiplatform 1.10.0 发布

JetBrains 在 2026 年 1 月正式发布 Compose Multiplatform 1.10.0，带来三个重大变化：

### 1. 统一 `@Preview` 注解
之前 Android/iOS/Desktop 有三套不同的 Preview 注解，现在统一为 `@Preview`。这意味着：
- **一次编写，多端预览** 的开发体验正式成为现实
- Compose Multiplatform 的生产可用性大幅提升

### 2. Navigation 3 稳定版
路由库正式进入 3.x 时代，跨平台导航 API 趋于成熟。

### 3. Stable Compose Hot Reload
热重载终于稳定，减少妈妈在调试 Compose UI 时的等待时间。
