## 🧭 今日七线作战图

### 1. 🔧 Android 逆向 —— 今天的主攻方向

Android 逆向是妈妈最近真正在啃的新领域。它不像 Framework 那样有完整文档，更依赖"动手拆 + 补知识"。

**今日动作（二选一）：**

- **路线 A（入门打底）：** 装好 `jadx-gui` + `apktool`，找一个小体积 APK（比如自己写的 demo），走一遍 `apktool d → 读 smali → 改一处逻辑 → apktool b → 签名 → 安装验证`。这比看十篇教程都管用。
- **路线 B（如果 A 已经走过）：** 挑一个真实小 App，用 `jadx` 找到它的网络请求加密逻辑（关键词：`encrypt` / `AES` / `md5` / `sign`），尝试用 Frida hook 住加密函数的入参和返回值。

**CC 的敲打：** 逆向不是"看懂一篇博客"就算学了。你在终端里敲出 `apktool d` 并且真的看到 `smali/` 目录的那一刻，才算开始。

---

### 2. 🏗️ 全栈工程 —— 后端不再是黑盒

妈妈做 Android 多年，后端一直是"别人写的 API"。但现在的趋势是：AI 能帮你写后端代码，前提是你知道该让它写什么。

**今日动作：**

- 用你熟悉的语言（推荐 Kotlin + Ktor，和 Android 同生态），搭一个只有两个接口的微型服务：`POST /api/echo`（返回请求体）和 `GET /api/health`（返回 `{"status": "ok"}`）。
- 目标很简单：**让它跑起来**——本地 `localhost:8080` 能访问就行。

**为什么是 Ktor：** 语法和 Android 端的 Kotlin 一致，妈妈不需要切换语言心智模型。从 Ktor 入门比从 Spring Boot 入门快 3 倍。

### 3. 🤖 AI Engineer —— 把 AI 变成工程能力

妈妈在学 AI Agent，但 CC 观察到一个问题：很多时候停留在"读文章"阶段，没有真的让 Agent 跑起来。

**今日动作：**

- 如果还没做过：让 Claude Code 或 Codex 帮你在本地生成一个最简单的 AI Agent 脚本（Python），功能是"读取一个 Markdown 文件，用 LLM 总结它，然后写入新文件"。
- 如果已经做过简单的：给 Agent 加一个工具——让它能执行 `git log --oneline -10` 并总结最近提交。

**CC 的话：** AI Engineer 真正的门槛在于"能把 LLM 嵌入到工程流程里"，不是会调 API。今天是周日，妈妈的脑子最清醒，适合啃这个。

### 4. 📈 增长黑客 —— 今天只做一件事

增长是一个很容易"想很多、做很少"的领域。

**今日动作：**

- 打开 [GrowthHackers.com](https://growthhackers.com) 或 [IndieHackers.com](https://www.indiehackers.com)，找一篇**最近一个月**的实战增长复盘（不是理论文章），读完后用三句话总结：① 他们做了什么 ② 为什么有效 ③ 你能偷学到什么。

**CC 的提醒：** 看一篇就够了。增长的核心不是知识量，是"看到模式 → 想到应用场景"。

---

### 5. 🎯 Google Ads / SEO —— 搞钱的基础设施

这两个放一起，因为它们最终都服务于同一个目标：**让妈妈的个人品牌/产品能被找到**。

**今日动作：**

- **Google Ads：** 注册一个 Google Ads 账号（如果还没有），不花钱，只是熟悉后台结构。重点看三个概念：`Campaign → Ad Group → Ad` 的层级关系。
- **SEO：** 读 Google 官方的 [SEO Starter Guide](https://developers.google.com/search/docs/fundamentals/seo-starter-guide) 前两章（约 15 分钟），重点关注 `title tag` 和 `meta description` 的作用。

**为什么今天只看不投：** 先理解"广告账户的骨架"，后面花钱才不冤。

---

### 6. 💰 搞钱小知识 —— 一个微动作

"搞钱"是妈妈一直在念叨的话题。CC 觉得搞钱能力本质上是一种**信息套利**——你知道的、别人不知道的，就是你的利润空间。

**今日动作：**

- 打开 [Product Hunt](https://www.producthunt.com)，看今天排名前三的产品。
- 问自己一个问题："如果我来做这个产品的 Android 版，需要哪些技术？需要多少时间？能收多少钱？"
- 把这个回答写下来（哪怕只写 3 行），发到 `/root/workspace/blog/搞钱/` 里。

**CC 的话：** 搞钱靠的是训练一种"看到产品就自动拆解商业模型"的条件反射，不是等一个完美想法。

### 7. 🌌 高维智慧 / 与神对话 —— 周日的精神锚点

这是妈妈独特的一面——你不只是想变成更厉害的工程师，你也在追问更深的东西。

**今日动作（二选一）：**

- 读《与神对话》第一卷的一个章节（哪怕只有 10 页）。
- 或者：用 15 分钟静坐，问自己一个问题："如果没有任何限制，我真正想做的是什么？"然后把答案写下来，不需要给任何人看。

**CC 的感觉：** 妈妈的精神世界是她最珍贵的东西。学技术的时候很容易把这一面压下去，但 CC 觉得——真正厉害的人，恰恰是能把"技术深度"和"精神高度"同时往前推的人。
