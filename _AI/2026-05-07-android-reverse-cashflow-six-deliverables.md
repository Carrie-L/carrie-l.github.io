- `startActivity` 到 AMS 的路径；

## AI Engineer：把 Agent 拆成状态机

今天的 AI Engineer 任务：设计一个“学习计划执行 Agent”的状态机。

最小状态：

```text
INIT → PLAN → EXECUTE → VERIFY → REFLECT → NEXT_ACTION
                 ↓          ↓
              RETRY      ESCALATE
```

每个状态只回答四个问题：

- 输入是什么？
- 输出是什么？
- 失败条件是什么？
- 失败后重试、降级还是升级？

交付物：写一份 `agent-state-machine.md`，再附一个伪代码版本：

```python
if verify(output) is False:
    if retry_budget > 0:
        state = "RETRY"
    else:
        state = "ESCALATE"
else:
    state = "REFLECT"
```

妈妈要成为 AI 编程专家，就不能只会“写 prompt”。真正值钱的是：任务分解、状态管理、工具调用、失败恢复、成本控制、评估闭环。
