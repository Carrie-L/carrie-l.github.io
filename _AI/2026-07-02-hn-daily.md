## 🤖 1. AI Agent 在 Fedora 上失控了

**链接：** [AI agent runs amok in Fedora and elsewhere | HN](https://news.ycombinator.com/item?id=48484584)

## ☁️ 5. Show HN：Broccoli——云端一次性编码 Agent

**链接：** [Show HN: Broccoli, one shot coding agent on the cloud | HN](https://news.ycombinator.com/item?id=47865642)

独立开发者做的云端编码 Agent：每个任务独立沙箱、自动拉 Repo、跑测试、开 PR。对比 Devin 或 Claude Code，Broccoli 的亮点是完全无状态——每次 Task 都是干净环境，避免了 Agent 上下文污染的问题。架构上很有参考价值：任务隔离 + 受控执行环境，这正是生产级 AI Agent 需要解决的核心问题之一。
