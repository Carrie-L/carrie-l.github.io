## 📖 逐語 基礎解析

### 1. 契約（けいやく）🔄
- **詞性**：名词
- **意味**：双方约定好的规则；合同
- **よく使う形**：契約を結ぶ（むすぶ）= 签合同 / 契約書（けいやくしょ）= 合同文件
- **N4 例句**：
  > 来月、新しいアパートの**契約**を結びます。
  > （下个月，要签新公寓的合同。）
- **CC の一言** 🍓：ママ、アパート借りるとき「契約」するよね。あの紙にサインするあれ、まさに「契約」！

### 2. 誤解（ごかい）🔄
- **詞性**：名词・する动词
- **意味**：理解错了；misunderstanding
- **よく使う形**：誤解する = 误解 / 誤解を解く（とく）= 解除误会
- **N4 例句**：
  > それは**誤解**です。私はそう言っていません。
  > （那是误会。我没那么说。）
- **CC の一言** 🏕️：「誤」は「まちがい」、「解」は「理解」。漢字で覚えると「間違った理解」→ 誤解！わかりやすいでしょ？

### 3. 活かす（いかす）
- **詞性**：他动词
- **意味**：让已有的能力/经验发挥出来；to utilize effectively
- **よく使う形**：経験を活かす = 活用经验 / 能力を活かす = 发挥能力
- **N4 例句**：
  > 日本語の勉強を**活かして**、日本の会社で働きたいです。
  > （想活用日语学习，在日本的公司工作。）
- **CC の一言** 🍊：「生かす」とも書くよ。でも「活かす」のほうが「活用する」ニュアンスが強い！

### 4. 事態（じたい）
- **詞性**：名词
- **意味**：正在发生的情况（通常不太好或很严重）
- **よく使う形**：深刻な事態 = 严重的事态 / 事態が悪化する = 事态恶化
- **N4 例句**：
  > この**事態**をすぐに社長に報告してください。
  > （请立刻把这个情况报告给社长。）
- **CC の一言** ⚠️：「事態」は「大変なことが起きている」ニュアンス。ただの「こと」じゃなくて、「やばいこと」！

### 5. 社会（しゃかい）
- **詞性**：名词
- **意味**：society
- **よく使う形**：社会人（しゃかいじん）= 上班族/社会人 / 社会問題 = 社会问题
- **N4 例句**：
  > 来年から**社会人**になります。
  > （明年开始我就是社会人了。）
- **CC の一言** 💼：N4で習ったけど、N2読解では「社会制度」「社会問題」みたいに複合語で出るから要注意！

### 6. 条件（じょうけん）
- **詞性**：名词
- **意味**：condition；做某事需要满足的前提
- **よく使う形**：条件を満たす = 满足条件 / 条件付き = 附带条件
- **N4 例句**：
  > この仕事の**条件**は、日本語N2レベルが必要です。
  > （这份工作的条件是，需要日语N2水平。）
- **CC の一言** 📝：「条」は「筋道」、「件」は「事柄」。筋道のある事柄 → 条件！

### 7. 水準（すいじゅん）
- **詞性**：名词
- **意味**：level; standard（比较正式）
- **よく使う形**：生活水準 = 生活水平 / 水準が高い = 水准高
- **N4 例句**：
  > 日本の教育**水準**はとても高いです。
  > （日本的教育水平很高。）
- **CC の一言** 📊：「レベル」でも通じるけど、新聞やレポートでは「水準」が好まれるよ。

### 8. 制度（せいど）
- **詞性**：名词
- **意味**：system; institution（社会或组织定下的规则体系）
- **よく使う形**：教育制度 = 教育制度 / 社会制度 = 社会制度
- **N4 例句**：
  > この国は医療**制度**がしっかりしています。
  > （这个国家的医疗制度很完善。）
- **CC の一言** 🏛️：「制」= コントロール、「度」= ものさし。社会をコントロールするものさし → 制度！

### 9. 成果（せいか）
- **詞性**：名词
- **意味**：努力之后得到的好结果；result; achievement
- **よく使う形**：成果を出す = 出成果 / 研究成果 = 研究成果
- **N4 例句**：
  > 毎日勉強した**成果**が出て、テストに合格しました。
  > （每天学习的成果出来了，考试合格了！）
- **CC の一言** 🎉：ママが毎日コツコツやってる日本語、まさにこれが「成果」に繋がるんだよ！

### 10. 政策（せいさく）
- **詞性**：名词
- **意味**：policy（政府或组织的方针）
- **よく使う形**：経済政策 = 经济政策 / 政策を決める = 决定政策
- **N4 例句**：
  > 新しい**政策**で、税金が安くなりました。
  > （因为新政策，税变便宜了。）
- **CC の一言** 🏛️：「政」= 政治、「策」= 計画。政治の計画 → 政策！

## ⚡ まぎらわしいペア（易混词）

| ペア | 違い |
|------|------|
| **契約 vs 条件** | 契約 = 双方签的「合同」；条件 = 要达成某事需要的「前提」 |
| **事態 vs 事情** | 事態 = 正在发生的（坏）情况；事情 = 背后的经过/原因 |
| **水準 vs レベル** | 水準 = 正式书面；レベル = 口语/日常 |
| **成果 vs 結果** | 成果 = 努力后的「好结果」；結果 = 中性，好坏都可能 |

## 🎬 ミニシーン 1：アパートの契約 🏠

（場所：不動産屋さん）

**店員**：では、こちらの**条件**でよろしいですか？家賃は8万円、敷金1ヶ月です。

**ママ**：はい。ただ、ペットを飼ってもいいという**条件**は入っていますか？

**店員**：はい、契約書に書いてあります。では来週、正式に**契約**を結びましょう。

**ママ**：あの、「2年以内に引っ越したら違約金がかかる」っていうのは…？

**店員**：あ、それは**誤解**です！その**制度**は去年の**政策**で変わりました。今は1年以内だけです。

**ママ**：なるほど！**誤解**していました。この**社会**ではそういう**制度**が変わったんですね。

> 🍓 **このシーンで練習した単語**：契約 / 条件 / 誤解 / 制度 / 政策 / 社会

## 🎬 ミニシーン 2：職場での成果発表 💼

（場所：会社の会議室）

**上司**：今回のプロジェクトの**成果**を報告してください。

**ママ**：はい。今回の**成果**として、売上が前年より20%上がりました。

**上司**：それは素晴らしい。でも、途中でトラブルがあったそうですね。どんな**事態**でしたか？

**ママ**：はい、システムが急に止まるという**事態**が起きました。でも、前の会社での経験を**活かして**、2日で復旧しました。

**上司**：なるほど、いい経験を**活かしました**ね。当社の技術**水準**も上がってきました。

**ママ**：ありがとうございます。次のプロジェクトでもこの経験を**活かしたい**です。

> 🍊 **このシーンで練習した単語**：成果 / 事態 / 活かす / 水準
