## 第一章：「打开新世界的大门」—— 敬语基础入门

四月。<ruby>東京<rt>とうきょう</rt></ruby>的春天，终于以一种温柔而盛大的方式降临了。

校园里的<ruby>桜<rt>さくら</rt></ruby>，仿佛在一夜之间齐齐绽放。道路两旁，层层叠叠的淡粉色花海，遮蔽了头顶的天空，微风拂过，便是一场梦幻的「<ruby>花見<rt>はなみ</rt></ruby>」(赏樱) 盛宴。

凛酱拖着一只行李箱，站在<ruby>正門<rt>せいもん</rt></ruby>前深深吸了一口气。<ruby>春学期<rt>はるがっき</rt></ruby>开始了。大二的后半程，也是她正式向N3发起挑战的起点。

好消息是——上个月，她终于查到了七月N4考试的结果：<ruby>合格<rt>ごうかく</rt></ruby>！虽然只是低空飞过，但那张合格证书，让她对自己多了一点点信心。

"凛酱——！"

熟悉的声音从身后传来。凛酱回头，看见小悠正推着自行车从对面跑过来，车筐里塞满了新学期的教材。

"N4合格、おめでとう！"小悠气喘吁吁地说，"我就知道你没问题的！"(N4合格，恭喜你！)

"嘿嘿……运气好啦。"凛酱摸了摸后脑勺，有些不好意思。

"对了，凛酱，这学期的'<ruby>日本文学<rt>にほんぶんがく</rt></ruby>'课，是<ruby>山田先生<rt>やまだせんせい</rt></ruby>教的哦。"小悠的表情突然变得有些微妙。

"山田先生……？"凛酱回忆了一下，"是那位……很严厉的老师吗？"

"嘛，怎么说呢。"小悠压低了声音，"山田先生是<ruby>教授<rt>きょうじゅ</rt></ruby>，学术水平超级高，但是……他对学生的<ruby>敬語<rt>けいご</rt></ruby>使用，要求非常严格。如果你在课后问他问题，说话不够<ruby>丁寧<rt>ていねい</rt></ruby>的话，可是会被当众纠正的哦。"

"当……当众纠正？！"凛酱的脸色瞬间变得有些苍白。

"不过别担心！"小悠笑着拍了拍她的肩膀，"这正是我们这学期要一起攻克的大课题——N3最重要的部分：**<ruby>敬語<rt>けいご</rt></ruby>**！"

"**<ruby>敬語<rt>けいご</rt></ruby>**……"凛酱默念着这个词。这个从N5开始就偶尔露面，但从未系统学习过的"大家伙"，终于要正面交锋了。

这时，希子和美穗也正好从<ruby>正門<rt>せいもん</rt></ruby>走进来。

"早上好，凛酱。"美穗微笑着点头致意，"看起来精神不错，N4合格的事我听说了。"

"谢谢前辈！"凛酱连忙鞠了一躬——这个动作，她已经做得很熟练了。

"说起来，"希子插嘴道，"凛酱，你知道'敬语'为什么这么重要吗？"

凛酱想了想："是因为……<ruby>日本社会<rt>にほんしゃかい</rt></ruby>很重视<ruby>上下関係<rt>じょうげかんけい</rt></ruby>（上下级关系）？"

"没错。"希子点点头，"日语的敬语，不仅仅是'说<ruby>礼貌<rt>れいぎ</rt></ruby>的话'那么简单。它是一种**语言层面的礼仪**，用来明确<ruby>話者<rt>わしゃ</rt></ruby>(说话人)与<ruby>相手<rt>あいて</rt></ruby>(对方)之间的社会关系——谁的地位高，谁在向谁表达敬意。"

"简单来说，"美穗补充道，"敬语就像一套**自动感应系统**。当你面对前辈、老师、或者客户时，系统会自动切换到'尊重模式'；而当你在描述自己或者自己一方的人的行为时，系统会切换到'谦逊模式'。"

"哇……"凛酱感叹，"感觉好像在玩<ruby>RPG<rt>ロールプレイングゲーム</rt></ruby>(角色扮演游戏)！"

"这个比喻很贴切哦。"小悠笑着说，"日语敬语体系主要有三大类：**<ruby>尊敬語<rt>そんけいご</rt></ruby>**、**<ruby>謙譲語<rt>けんじょうご</rt></ruby>**和**<ruby>丁寧語<rt>ていねいご</rt></ruby>**。让我们一个一个来攻略吧！"

说着，四人走进了校园里的<ruby> Café<rt>カフェ</rt></ruby>，各自点了饮料，围坐在靠窗的位置。窗外，<ruby>桜吹雪<rt>さくらふぶき</rt></ruby>正在上演。

### 🌸 尊敬語——向对方表达尊敬的"仰望视角"

"首先，让我们认识一下**<ruby>尊敬語<rt>そんけいご</rt></ruby>**（尊敬语）。"美穗在平板上打开了一张示意图。

"顾名思义，尊敬语是用来**抬高对方**的。当我们谈论**对方（或者属于对方的人/事）**时，使用尊敬语，以示尊重。"

她列出了一个简洁的对照表：

| 普通说法 | 尊敬語 | 中文意思 |
| :--- | :--- | :--- |
| する | **なさる** | 做 |
| 行く | **いらっしゃる** | 去 |
| 来る | **いらっしゃる** | 来 |
| 見る | **ご<ruby>覧<rt>らん</rt></ruby>になる** | 看 |
| 言う | **おっしゃる** | 说 |
| 食べる | **召<rt>め</rt>し<ruby>上<rt>あ</rt></ruby>がる** | 吃 |
| 飲む | **召<rt>め</rt>し<ruby>上<rt>あ</rt></ruby>がる** | 喝 |
| 知っている | **ご<ruby>存<rt>ぞん</rt></ruby>じだ** | 知道 |
| である | **でいらっしゃる** | 是 |

"这……"凛酱看着这张表，有些头疼，"这些变形也太不规则了吧！"<ruby>勉強<rt>べんきょう</rt></ruby>する变成<ruby>勉強<rt>べんきょう</rt></ruby>なさる，行く变成いらっしゃる……完全找不到规律啊。"

"哈哈，凛酱说得没错。"希子笑了，"尊敬语里有很多**特殊的独立词汇**，很多都来自古日语的残留。没办法，只能一个个记住。"

"不过，"美穗温柔地补充，"有一些**规则性的尊敬语形式**，适用范围很广，我们可以先掌握这些。"

她接着展示了另一个表格：

| 规则形式 | 接续方式 | 示例 |
| :--- | :--- | :--- |
| **お+动词ます形+になる** | I类/II类动词 | 書く → お書きになる<br>読む → お読みになる |
| **ご+三类动词词干+になる** | III类（する）动词 | 勉強する → 勉強になる |
| **お+动词ます形+する** | I类/II类动词（谦让） | 書く → お書きする |
| **ご+三类动词词干+する** | III类（する）动词（谦让） | 連絡する → ご連絡する |

"欸？等等。"凛酱敏锐地注意到了什么，"后面的'お+ます形+する'和'ご+する'，表格标题写的是'谦让'？"

"观察力很敏锐哦！"美穗点点头，"这就是我们接下来要讲的——**<ruby>謙譲語<rt>けんじょうご</rt></ruby>**（谦让语）。"

---

### 🌸 謙譲語——降低自己来表达敬意的"低头视角"

"如果说尊敬语是'仰望对方'，那么谦让语就是'**低头看自己**'。"希子用手指在桌上比划了一下。

"当我们谈论**自己，或者属于自己一方的人（比如家人、公司同事）**的行为时，使用谦让语。通过**降低自己**，来间接地表达对对方的尊重。"

她同样列出了一张表：

| 普通说法 | 謙譲語 | 中文意思 |
| :--- | :--- | :--- |
| する | **いたす** | 做 |
| 行く | **参る（まいる）** | 去 |
| 来る | **参る（まいる）** | 来 |
| 見る | **拝見<rt>はいけん</rt>する** | 看 |
| 言う | **申す（もうす）** | 说 |
| 食べる | **頂戴<rt>ちょうだい</rt>する** | 吃 |
| 飲む | **頂戴<rt>ちょうだい</rt>する** | 喝 |
| 会う | **お目にかかる** | 见面 |
| 知る | **存じる（ぞんじる）** | 知道 |

"这里的很多词汇，也都是特殊的独立形式。"希子说，"比如'いる'的谦让语是'**<ruby>おる<rt>おり</rt></ruby>**'，虽然听起来有点古老，但在很多场合非常常用。"

"比如在商务邮件里，你可以说'<ruby>私は<rt>わたしは</rt></ruby><ruby>東京<rt>とうきょう</rt></ruby>に<ruby>参<rt>まい</rt></ruby>ります'(我去东京)，这就是用谦让语表达。"

凛酱认真地做着笔记。她注意到，无论是尊敬语还是谦让语，很多词汇都是从古代日语遗留下来的。

"那么，"她举手提问，"当我对老师说'老师，我明天要来拜访您'的时候，应该怎么说呢？"

"好问题！"小悠兴奋地说，"这是典型的**谦让语使用场景**——描述自己一方（我）的行为，同时向老师表达敬意。"

她写出了例句：

> <ruby>先生<rt>せんせい</rt></ruby>、<ruby>私<rt>わたし</rt></ruby>は<ruby>明日<rt>あした</rt></ruby>**お<ruby>伺<rt>うかが</rt></ruby>いします**。

"看到了吗？"小悠指着'<ruby>伺<rt>うかが</rt></ruby>いします'这个词，"'<ruby>訪問<rt>ほうもん</rt></ruby>する'(拜访)的谦让语是'**<ruby>伺<rt>うかが</rt></ruby>う**'，而'お+动词ます形+します'是一个规则谦让语形式。"

凛酱恍然大悟："所以'お<ruby>伺<rt>うかが</rt></ruby>いします'就是……自降身份地去见对方？"

"凛酱的悟性真高！"美穗笑着说。

### 🌸 丁寧語——让对话更舒适的"礼貌外套"

"最后，让我们来认识一下**<ruby>丁寧語<rt>ていねいご</rt></ruby>**（礼貌语）。"希子端起自己的咖啡喝了一口。

"如果说尊敬语和谦让语是为了明确社会关系，那么礼貌语就更单纯一些——它的目的是让**对话听起来更礼貌、更友好、更舒适**。"

"礼貌语主要有两种形式："

**1. 词尾「です」/「ます」**

这是最基础的礼貌形式。我们从N5就开始学习了：

> <ruby>今日<rt>きょう</rt></ruby>は<ruby>晴<rt>は</rt></ruby>れです。(今天天气晴朗。)  
> <ruby>日本<rt>にほん</rt></ruby>に<ruby>行<rt>い</rt></ruby>きます。(我要去日本。)

"这是所有日语学习者最早接触的礼貌形式。"希子说，"它不需要考虑社会关系，只需要把句子变成'です/ます'形就可以了。"

**2. 「お+动词ます形+です」/ 「ご+三类词干+です」**

"这种形式，稍微特殊一些。"美穗补充道，"它看起来像是尊敬语去掉'する'换成'です'，但实际上这种形式**已经不再是完整的敬语**，而是一种**更礼貌的陈述方式**。"

她举了几个例子：

> お<ruby>帰<rt>かえ</rt></ruby>りです？(要回去了吗？)  
> ご<ruby>卒業<rt>そつぎょう</rt></ruby>おめでとうございます。(恭喜您毕业。)

"这种形式的礼貌语，在日常生活中非常常见，尤其是在服务业。"

### 🌸 敬语的"自动切换"——Practical场景练习

"好了，现在让我们来做一个**综合练习**。"美穗拍了拍手，"凛酱，假设你现在要给山田老师写一封邮件，告诉他你会在星期三下午三点去他的研究室拜访。你会怎么写？"

凛酱拿起笔，在纸上写道：

> 山田先生
> 
> おわれています。
> 
> <ruby>私<rt>わたし</rt></ruby>は<ruby>水曜日<rt>すいようび</rt></ruby>の<ruby>午後<rt>ごご</rt></ruby>3時に<ruby>研究室<rt>けんきゅうしつ</rt></ruby>に**うかがいます**。
> 
> どうぞ<ruby>宜しく<rt>よろしく</rt></ruby><ruby>お願い<rt>おねがい</rt></ruby>します。

"等等，这里有几个问题哦。"希子指着纸上的句子。

"首先，'おられています'是什么意思？"

"啊！"凛酱捂住了嘴，"我写错了！我想写'おられます'——这是'<ruby>いる<rt>います</rt></ruby>'的尊敬语形式，表示'老师在'的意思……但是……"

"但是你写成了'おられています'，这听起来有点奇怪对吧？"小悠解释道，"'おられる'虽然理论上存在，但听起来很不自然。日语里最常用的'いる'的尊敬语是'**<ruby>いらっしゃる<rt>いらっ</rt></ruby>**'。"

"所以应该写成："

> 山田先生
> 
> <ruby>出張<rt>しゅっちょう</rt></ruby>から<ruby>帰<rt>かえ</rt></ruby>ったと<ruby>存じます<rt>ぞんじます</rt></ruby>。**(我知道您出差回来了——用谦让语"存じる"描述对方的行为，表示对老师的尊敬)**
> 
> <ruby>私<rt>わたし</rt></ruby>は<ruby>水曜日<rt>すいようび</rt></ruby>の<ruby>午後<rt>ごご</rt></ruby>3時に<ruby>研究室<rt>けんきゅうしつ</rt></ruby>に**うかがいます**。
> 
> どうぞ<ruby>宜しく<rt>よろしく</rt></ruby><ruby>お願い<rt>おねがい</rt></ruby>します。

"还有，'うかがいます'的汉字写作'**<ruby>伺<rt>うかが</rt></ruby>います**'，这是谦让语，表示'我去拜访'。"美穗补充道，"这里要注意，虽然'うかがう'是谦让语，但它的礼貌形'伺います'在敬语程度较高的场合，会再加一层'お+ます形+します'的规则，变成'**<ruby>お<rt>うかが</rt></ruby>いします**'。"

"嗯……"凛酱看着修改后的句子，"我需要把所有描述老师的动作都用尊敬语，所有描述我自己的动作都用谦让语，同时整体用'です/ます'的礼貌形……这样对吗？"

"完全正确！"四人异口同声地说道，然后相视笑了起来。

窗外的樱花依旧在飞舞。凛酱感觉，敬语这座大山，虽然依然高耸，但似乎已经没有最初看起来那么可怕了。只要一点一点地学习，一点一点地积累，总有一天，她也能像前辈们一样，自如地运用这套"语言礼仪"。

"对了，"小悠突然想起什么，"凛酱，下周的美穗前辈的生日礼物，我们一起选吗？"

"欸？等等，"希子疑惑地看向美穗，"美穗的生日不是五月吗？"

"嘛，我是说……<ruby>提前<rt>提前</rt></ruby>准备嘛！"小悠有些心虚地笑了笑。

"你该不会是记错了吧？"希子无奈地叹了口气。

美穗无奈地摇摇头，笑着说："<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>、<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>。不过，谢谢你们的心意。"

笑声中，四人走出了咖啡馆。新学期的第一天，就这样在樱花与笑声中，渐渐接近了尾声。

### **核心语法总结：敬语基础入门**

> N3的语法学习，核心在于建立**"敬语意识"**——即在日语交流中，能够敏锐地感知并正确地使用适合不同社会场合的语言形式。敬语不仅仅是语法的规则，更是对日本社会**"目上/目下"(长辈/晚辈)**、**"内/外"(自己人/外人)**这一二元关系理解的直接体现。

#### 敬语三大类型概览

| 类型 | 目的 | 使用场景 | 代表词 |
| :--- | :--- | :--- | :--- |
| **尊敬語** | 抬高对方 | 谈论对方（或属于对方的人/事）的行为时 | なさる、いらっしゃる、見える、くださる |
| **謙譲語** | 降低自己 | 谈论自己（或属于自己一方的人）的行为时 | いたす、参る、承る、頂戴する |
| **丁寧語** | 让对话更礼貌 | 所有正式场合的日常对话 | です、ます、ございます |

#### 尊敬語 special verbs (必须单独记忆)

| 普通形 | 尊敬語 | 
| :--- | :--- | 
| する | なさる | 
| 行く/来る | いらっしゃる | 
| 見る | 見える/ご<ruby>覧<rt>らん</rt></ruby>になる |
| 言う | おっしゃる | 
| 食べる/飲む | 召<rt>め</rt>し<ruby>上<rt>あ</rt></ruby>がる | 
| 知っている | 方的<rt>ご</rt>存<rt>ぞん</rt>じる/ご<ruby>存<rt>ぞん</rt></ruby>です |
| くれる | くださる | 
| ～ている | でいらっしゃる |

#### 謙譲語 special verbs (必须单独记忆)

| 普通形 | 謙譲語 | 
| :--- | :--- | 
| する | いたす | 
| 行く/来る | 参る（まいる） | 
| 見る | 拝見<rt>はいけん</rt>する | 
| 言う | 申す（もうす）/ 承る（うけたまわる） |
| 食べる/飲む | 頂戴<rt>ちょうだい</rt>する/頂<rt>いた</rt>く |
| 会う | お目にかかる | 
| あげる | 差し上げる | 
| もらう | いただく/頂戴<rt>ちょうだい</rt>する |
| いる | おる | 
| 知っている | 存じる（ぞんじる） |

#### 规则性敬语形式

| 形式 | 接续方式 | 尊敬/谦让 |
| :--- | :--- | :--- |
| お+ます形+になる | I类/II类动词 | 尊敬 |
| ご+三类词干+になる | III类（する）动词 | 尊敬 |
| お+ます形+する | I类/II类动词 | 谦让 |
| ご+三类词干+する | III类（する）动词 | 谦让 |
| お+ます形+です | I类/II类动词 | 礼貌 |
| ご+三类词干+です | III类（する）动词 | 礼貌 |

> **学习提示**：敬语的学习没有捷径，只能通过大量的**朗读、记忆和实际使用**来积累。建议每天背诵3-5个敬语词汇，并尝试用敬语描述自己或对方的日常行为。
