  「Android Handler 消息机制」

  - WHAT：Handler + Looper + MessageQueue 三件套，是 Android 线程通信的核心
  - WHY：UI 只能在主线程更新，子线程完成任务后需要通过 Handler
  把结果"传回"主线程
  - HOW：子线程 handler.post { } → 消息进入主线程 MessageQueue → Looper 轮询取出
   → Handler 执行

  // 主线程创建Handler
  val handler = Handler(Looper.getMainLooper())

  // 子线程中切回主线程
  thread {
      val result = doHeavyWork()
      handler.post { textView.text = result }
  }
