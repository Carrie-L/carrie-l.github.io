## 🧮 算法

### 贪心算法 - 买卖股票最佳时机 II

**题目**：LeetCode 122 - 买卖股票的最佳时机 II
**思路**：只要明天比今天贵，今天买明天卖
**代码**：
```cpp
sum = 0; 
for(int i = 0; i < prices.size() - 1; i++) {
    if(prices[i+1] > prices[i]) {
        sum += prices[i+1] - prices[i];
    }
}
```
